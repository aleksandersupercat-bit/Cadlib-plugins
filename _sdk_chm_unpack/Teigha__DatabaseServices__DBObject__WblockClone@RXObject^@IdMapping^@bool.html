Параметры
Параметры
Описание
ownerPointer
Входной объект, который клонируется. Тип RxObject
idMap
Карта преобразований Id текущего объекта. Тип IdMapping
isPrimary
Признак, является ли объект первичным или имеет владельца. Тип bool
Возвращает

Возвращает созданной клон. Тип DBObject Если операция клонирования завершается ошибкой, возвращается значение null.

Описание

Этот метод практически идентичен методу DBObject.DeepClone(), за исключением того, что использование WblockCloneFiler не должно выходить за границы базы данных. Субклонирование базируется на жестком задании владельца и жестком указании на связи.
