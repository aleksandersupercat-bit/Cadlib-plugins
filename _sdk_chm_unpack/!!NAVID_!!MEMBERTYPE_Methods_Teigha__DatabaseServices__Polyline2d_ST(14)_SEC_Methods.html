Методы
Имя
Описание
AppendVertex
Метод добавляет новую вершину Vertex2d в список вершин полилинии.
ConvertToPolyType
Метод предназначен для преобразования полилинии в тип, указанный значением аргумента.
CurveFit
Метод создает гладкую кривую, состоящую из пар дуг, соединяющих каждую пару вершин. Эта операция эквивалентна коменде PEDIT с опцией "Fit".
GetEnumerator
Метод предназначен для получения перечислителя коллекции.
InsertVertexAt
Это обзор перегрузок InsertVertexAt.
NonDBAppendVertex
Метод добавляет объект Vertex2d в список вершин полилинии и делает полилинию владельцем вершины. Полилиния не должна быть резидентной в базе данных.
SplineFit
Это обзор перегрузок SplineFit.
Straighten
Метод удаляет из полилинии все вершины сплайна и аппроксимирующей кривой и определяет все оставшиеся вершины в качестве простых вершин. Эта операция эквивалентна команде PEDIT с опцией "Decurve".
VertexPosition
Метод предназначен для получения координат вершины в мировой системе координат (WCS). Метод извлекает координаты X и Y из вершины, а координату Z - из полилинии, после чего применяет преобразование из системы координат полилинии (OCS) в WCS для получения позиции в мировой системе координат.
