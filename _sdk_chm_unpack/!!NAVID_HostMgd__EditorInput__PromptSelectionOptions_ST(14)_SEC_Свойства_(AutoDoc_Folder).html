HostMgd.EditorInput.PromptSelectionOptions  Условные обозначения
PromptSelectionOptions Свойства
Имя
Описание
AllowDuplicates
Свойство определяет, разрешено ли добавление дублирующихся объектов в результирующий набор выбора.
AllowSubSelections
Свойство определяет, разрешён ли выбор подмножества объектов (например, вершин, рёбер или граней) внутри составных объектов, таких как полилинии, блоки или 3D-тела.
ForceSubSelections
Свойство определяет, будет ли выбор ограничен только подмножествами объектов — например, сегментами полилинии или элементами блока.
Keywords
Свойство предоставляет доступ к коллекции ключевых слов, доступных для выбора при операции ввода.
MessageForAdding
Свойство возвращает или устанавливает сообщение, отображаемое при добавлении объектов в выборку.
MessageForRemoval
Свойство возвращает или устанавливает сообщение, отображаемое при удалении объектов из текущего набора выбора.
PrepareOptionalDetails
Свойство устанавливает или возвращает режим N (No details), определяя, следует ли пропускать подготовку дополнительных данных о выбранных объектах.
RejectObjectsFromNonCurrentSpace
Свойство определяет, будут ли отклоняться объекты, находящиеся вне текущего пространства (модель или лист), при операции выбора.
RejectObjectsOnLockedLayers
Свойство определяет, разрешён ли выбор объектов, расположенных на заблокированных слоях.
RejectPaperspaceViewport
Свойство определяет, будет ли отклоняться выбор видовых экранов в пространстве листа (Paper Space).
SelectEverythingInAperture
SingleOnly
Свойство устанавливает режим :S для функции HostSSGet, ограничивая выбор только одним объектом.
SinglePickInSpace
Свойство включает режим :A для HostSSGet, позволяя выполнять выбор объектов только по одиночному щелчку без открытия рамки или окна выбора.
