Ниже перечислены события класса Editor.

События
Имя
Описание
Dragging
Событие возникает перед началом операции перетаскивания объекта в чертеже. Событие не реализовано.
DraggingEnded
Событие возникает при завершении пользователем операции перетаскивания объекта в чертеже. Событие не реализовано.
EnteringQuiescentState
Событие возникает, когда редактор переходит в состояние покоя — то есть пользователь завершил все действия и ожидает нового ввода.
LeavingQuiescentState
Событие возникает, когда пользователь выходит из состояния покоя и начинает новую активность в редакторе.
PointFilter
Событие возникает при перемещении мыши в окне редактора nanoCAD и позволяет фильтровать или изменять координаты указателя. Событие не реализовано.
PointMonitor
Событие возникает при каждом перемещении мыши в окне редактора nanoCAD, позволяя отслеживать координаты указателя в реальном времени.
PromptForEntityEnding
Событие возникает после завершения операции выбора геометрического объекта, но до фиксации результата. Событие не реализовано.
PromptForSelectionEnding
Событие возникает после завершения операции выбора объектов, но перед фиксацией результата. Событие не реализовано.
PromptedForAngle
Событие возникает после завершения операции ввода угла пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForCorner
Событие возникает после завершения операции ввода точки, используемой как угол прямоугольной области. Событие не реализовано.
PromptedForDistance
Событие возникает после завершения интерактивного ввода расстояния пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForDouble
Событие возникает после завершения интерактивного ввода вещественного числа пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForEntity
Событие возникает после завершения операции выбора геометрического объекта пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForInteger
Событие возникает после завершения операции ввода целого числа пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForKeyword
Событие возникает после завершения операции ввода ключевого слова пользователем в редакторе nanoCAD. Событие не реализовано.
PromptedForNestedEntity
Событие возникает после завершения операции выбора вложенного геометрического объекта — например, внутри блока или внешней ссылки. Событие не реализовано.
PromptedForPoint
Событие возникает после завершения операции ввода точки пользователем в редакторе nanoCAD.
PromptedForSelection
Событие возникает после завершения операции выбора объектов в редакторе nanoCAD, предоставляя доступ к набору выбранных элементов. Событие не реализовано.
PromptedForString
Событие возникает после завершения операции ввода строки пользователем в редакторе nanoCAD. Событие не реализовано.
PromptingForAngle
Событие возникает при начале интерактивной операции ввода угла в редакторе nanoCAD, позволяя модифицировать параметры ввода перед его началом. Событие не реализовано.
PromptingForCorner
Событие возникает при начале операции ввода точки, используемой как угол прямоугольной области. Событие не реализовано.
PromptingForDistance
Событие возникает при начале интерактивной операции ввода расстояния, позволяя настроить параметры команды до её запуска. Событие не реализовано.
PromptingForDouble
Событие возникает при начале интерактивной операции ввода вещественного числа, позволяя настроить параметры команды до её запуска. Событие не реализовано.
PromptingForEntity
Событие возникает при начале операции выбора геометрического объекта и позволяет настроить параметры команды выбора до её запуска. Событие не реализовано.
PromptingForInteger
Событие возникает при начале операции ввода целого числа, позволяя настроить параметры команды до её запуска. Событие не реализовано.
PromptingForKeyword
Событие возникает при начале операции ввода ключевого слова, позволяя настроить доступные варианты и сообщение приглашения. Событие не реализовано.
PromptingForNestedEntity
Событие возникает при начале операции выбора вложенного геометрического объекта, например, внутри блока или внешней ссылки. Событие не реализовано.
PromptingForPoint
Событие возникает при начале операции ввода точки и позволяет настроить параметры команды до её запуска.
PromptingForSelection
Событие возникает при начале операции выбора объектов и позволяет настроить параметры команды выборки до её запуска. Событие не реализовано.
PromptingForString
Событие возникает при начале операции ввода строки, позволяя настроить параметры приглашения до начала ввода. Событие не реализовано.
Rollover
Событие возникает при наведении курсора на графический примитив, вызывая его подсветку и позволяя выполнить действие на основе объекта под курсором. Событие не реализовано.
SelectionAdded
Событие возникает при добавлении одного или нескольких объектов в текущий набор выделенных объектов в редакторе nanoCAD.
SelectionRemoved
Событие возникает при удалении одного или нескольких объектов из текущего набора выделенных объектов в редакторе nanoCAD.
