Описание

Класс предназначен для хранения определений блоков в базе данных чертежа. 

Каждый блок определен записью BlockTableRecord, которая является вхождением коллекции BlockTable (таблицы блоков). В таблицу входят два типа блоков: служебные и пользовательские. 

Служебные блоки создаются платформой для собственных нужд (листы, представления размеров, штриховки и пр.). 

Пользовательские блоки создаются пользователями через интерфейс или с помощью API nanoCAD для повторяющихся фрагментов чертежа. 

Все примитивы чертежа хранятся в таблице блоков: либо в служебных блоках, либо в пользовательских. 

Также у всех блоков есть графическая часть BlockReference, которая ссылается на неграфическую. 

Графическая часть пользовательского блока всегда входит в какой-либо служебный блок, представляющий собой, например, лист документа.

Примечания

Этот .NET класс соответствует классу OdDbBlockTable nanoCAD API, который представляет таблицу символов для записи BlockTableRecord, которая представляет определение блока в чертеже базы данных.

Пример

В качестве примера работы с таблицей блоков приведена команда BlockTableReading, которая открывает таблицу блоков и перебирает её содержимое, выводя в команданую строку название каждого блока и его тип. 

 

C#
Копировать Код
public partial class Commands { [Teigha.Runtime.CommandMethod("BlockTableReading")] public void BlockTableUse() { // Получаем ссылки на активный документ HostMgd.ApplicationServices.Document doc = HostMgd.ApplicationServices.Application.DocumentManager.MdiActiveDocument; // Получаем ссылки на редактор докумена HostMgd.EditorInput.Editor ed = doc.Editor; // Получаем ссылки на базу данных документа Teigha.DatabaseServices.Database db = doc.Database; // Начало транзакции с базой данных документа using (Teigha.DatabaseServices.Transaction trans = db.TransactionManager.StartTransaction()) { // Открываем таблицу блоков для чтения Teigha.DatabaseServices.BlockTable blockTable = trans.GetObject(db.BlockTableId, Teigha.DatabaseServices.OpenMode.ForRead) as Teigha.DatabaseServices.BlockTable; // Перебираем таблицу блоков в цикле foreach(Teigha.DatabaseServices.ObjectId btr in blockTable) { // Открываем каждую запись таблицы Teigha.DatabaseServices.BlockTableRecord obj = trans.GetObject(btr, Teigha.DatabaseServices.OpenMode.ForRead) as Teigha.DatabaseServices.BlockTableRecord; // Выводим в командную строку ed.WriteMessage($"\nTable record name and type: {obj.Name} , {obj.GetType()} "); } trans.Abort(); } } }
