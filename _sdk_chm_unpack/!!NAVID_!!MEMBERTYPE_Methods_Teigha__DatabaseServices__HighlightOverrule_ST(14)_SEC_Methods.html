Методы
Имя
Описание
Highlight
Метод реализует подсвечивание указанного пути к подобъекту в данном объекте чертежа.
SetCustomFilter
Включает режим использования метода IsApplicable базового класса Overrule, чтобы установить допустимость применения объектных переопределений.
SetExtensionDictionaryEntryFilter
Метод включает режим поиска по словарю расширений для фильтрации объектов, к которым будут применены переопределения.
SetIdFilter
Метод включает режим фильтрации по ObjectId при поиске объектов, к которым будут применены переопределения. Переопределение будет вызвано только для объектов, чьи id присутствуют в заданном списке идентификаторов.
SetNoFilter
Отключает при переопределении использование всех методов предварительной фильтрации объектов (SetCustomFilter, SetExtensionDictionaryEntryFilter, SetIdFilter, SetXDataFilter), а также метода IsApplicable() базового класса Overrule.
SetXDataFilter
Метод включает предварительную фильтрацию объектов по расширенным данным (xdata) прежде чем применить переопределения.
Unhighlight
Отменяет в данном объекте подсвечивание указанного пути к подобъекту.
