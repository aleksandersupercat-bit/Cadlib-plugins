Метод задает единицы измерения на бумаге, используемые указанным объектом PlotSettings для полей, смещений, размера бумаги и единиц чертежа.
SetPlotRotation
Метод задает поворот при печати в настройках этого объекта PlotSettings.
SetPlotType

SetStdScale
Метод задает для указанного объекта PlotSettings стандартный масштаб как действительное число с плавающей точкой.
SetStdScaleType
Метод позволяет задать для указанного объекта PlotSettings стандартный масштаб из перечисления StdScaleType.
SetUseStandardScale
Метод позволяет контролировать использование стандартного масштаба для указанного объекта PlotSettings.
SetZoomToPaperOnUpdate
