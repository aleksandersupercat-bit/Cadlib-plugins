Spline Свойства Свойства
Spline Свойства  Условные обозначения
Свойства
Имя
Описание
Degree
Свойство Degree представляет степень данного сплайна.
EndFitTangent
Свойство EndFitTangent представляет касательный вектор в конечной определяющей точке.
FitData
Свойство FitData представляет данные подгонки этого сплайна.
FitTolerance
Свойство FitTolerance представляет точность аппроксимации для данного сплайна.
HasFitData
Свойство HasFitData позволяет выяснить, был ли данный сплайн создан методом определяющих точек.
IsNull
Свойство IsNull позволяет проверить, что у данного сплайна нет управляющих вершин.
IsPlanar
Свойство IsPlanar позволяет проверить, что данный сплайн является плоским.
IsRational
Свойство IsRational позволяет проверить, является ли данный сплайн рациональным.
NumControlPoints
Свойство NumControlPoints представляет количество управляющих вершин у данного сплайна.
NumFitPoints
Свойство NumFitPoints представляет количество определяющих точек у данного сплайна.
NurbsData
Свойство NurbsData представляет данные NURBS этого сплайна, включающие узлы, значения веса, значения допуска для узлов и управляющих вершин точек и т.д.
StartFitTangent
Свойство StartFitTangent представляет касательный вектор в начальной определяющей точке.
