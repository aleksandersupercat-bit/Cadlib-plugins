Teigha.DatabaseServices.CompoundObjectId
Имя
Описание
FullPath
Свойство предоставляет полный путь, т.е. путь вхождений BlockReference, концом которого является идентификатор ObjectId.
IsEmpty
Свойство позволяет выяснить, содержит ли данные этот объект CompoundObjectId.
IsExternal
Свойство позволяет проверить, является ли объект CompoundObjectId внешней ссылкой.
IsSimpleObjectId
Свойство позволяет проверить отсутствие пути, когда имеется только идентификатор конечного объекта (который может быть нулевым).
LeafId
Свойство обеспечивает доступ к идентификатору конечного ObjectId в BlockTableRecord, на который ссылается последнее вхождение BlockReference в пути, или просто ObjectId, если путь отсутствует.
Path
Свойство предоставляет путь вхождений блоков BlockReference.
TopId
Свойство предоставляет первый ObjectId в цепочке из вхождений блоков BlockReference или ObjectId конечного узла, если путь отсутствует.
Transform
Свойство предоставляет составное преобразование координат конечного объекта последовательности вхождений блоков в пространство модели, объединяя все преобразования BlockReference, от самого вложенного до самого верхнего.
