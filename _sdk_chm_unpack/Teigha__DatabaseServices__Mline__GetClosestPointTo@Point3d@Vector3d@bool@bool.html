Описание

Метод позволяет найти точку на объекте MLine, которая является ближайшей к проекции заданной точки givenPoint на плоскость данной мультилинии. 

Сначала точка givenPoint проецируется вдоль вектора normal на плоскость, содержащую объект MLine. Затем спроецированная точка используется для поиска ближайшей точки на объекте MLine. Если аргумент extend равен true, то конечные элементы объекта MLine будут «виртуально» продлены для поиска ближайшей точки. Если аргумент excludeCaps равен true, то любые торцевые ограничители и обозначения стыков сегментов объекта MLine будут игнорироваться.
