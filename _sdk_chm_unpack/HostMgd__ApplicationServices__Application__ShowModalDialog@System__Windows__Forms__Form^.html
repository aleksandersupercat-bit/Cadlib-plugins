Описание

Метод показывает модальный диалог WinForms. Необходимо пользоваться именно этим методом вместо метода Form.ShowDialog(), который может привести к непредсказуемому результату. При исползовании метода, достигается согласованность с редактором приложения: при запросе данных с чертежа (напрмер, точки или объекта) через диалоговое окно, оно пропадает, потом снова появляется, когда данные получены.
