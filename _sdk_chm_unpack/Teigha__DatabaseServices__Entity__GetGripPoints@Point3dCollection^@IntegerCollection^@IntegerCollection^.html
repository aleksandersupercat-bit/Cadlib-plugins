Метод GetGripPoints реализует поддержку "старого стиля" работы с ручками (точками управления) и предназначен для добавления координат ручек текущего геометрического объекта в уже существующий массив gripPoints.

Описание

Использование метода 

Этот метод поддерживает "старый стиль" ручек, в приложениях, созданных до улучшения функциональности работы с ручками. 

Разработка метода в производных классах 

Этот метод должен добавить к массиву gripPoints те точки, которые рассматриваются как координаты ручек для геометрического объекта. 

Массив gripPoints должен быть передан для всех геометрических объектов, включенных в операцию с ручками, так что массив может уже содержать координаты ручек для других геометрических объектов. Поэтому, является очень важным добавление новых точек в массив вместо назначения им уже существующих элементов массива. 

Параметры snapModes и geometryIds в настоящее время не используются.
