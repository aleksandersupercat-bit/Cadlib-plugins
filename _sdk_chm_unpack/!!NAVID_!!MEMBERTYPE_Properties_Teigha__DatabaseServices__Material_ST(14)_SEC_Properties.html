Material Свойства  Условные обозначения
Свойства
Имя
Описание
Ambient
Свойство представляет теневой компонент рассеянного цвета данного материала.
Anonymous
Свойство позволяет проверить, является ли этот материал «анонимным». Только для внутреннего использования.
Bump
Свойство представляет компонент рельефности данного материала.
ChannelFlags
Свойство представляет флаги канала для данного материала и позволяет настраивать требуемые комбинации каналов.
ColorBleedScale
Свойство, которое представляет коэффициент цветового заполнения для данного материала.
Description
Свойство представляет описание данного материала.
Diffuse
Свойство представляет основной диффузный компонент этого материала.
FinalGather
Свойство представляет режим финальной сборки для данного материала.
GlobalIllumination
Свойство представляет текущий режим общего освещения для этого материала.
IlluminationModel
Свойство представляет модель освещения, которая используется для этого материала в расширенном процессе рендеринга.
IndirectBumpScale
Свойство представляет коэффициент непрямой рельефности данного материала.
Luminance
Свойство представляет значение яркости материала.
LuminanceMode
Свойство представляет режим яркости данного материала.
Mode
Свойство представляет режим представления данного материала.
Name
Свойство представляет наименование данного материала.
NormalMap
Свойство представляет компонент NormalMap этого материала.
Opacity
Свойство представляет компонент непрозрачности этого материала.
ReflectanceScale
Свойство представляет коэффициент отражения данного материала.
Reflection
Свойство представляет компонент отражения данного материала.
Reflectivity
Свойство представляет степень отражающей способности этого материала.
Refraction
Свойство представляет компонент преломления данного материала.
SelfIllumination
Свойство представляет значение самосвечения данного материала.
Specular
Свойство представляет компонент зеркального отражения этого материала.
Translucence
Свойство представляет полупрозрачность этого материала.
TransmittanceScale
Свойство представляет коэффициент пропускания данного материала.
TwoSided
Свойство позволяет указать или проверить, что материал является двусторонним.
