MText Свойства Свойства
MText Свойства  Условные обозначения
Свойства
Имя
Описание
ActualHeight
Свойство возвращает фактическую высоту данного объекта MText (зависит от содержимого).
ActualWidth
Свойство возвращает фактическую ширину габаритного бокса для данного объекта MText (зависит от содержимого).
AlignChange
Свойство представляет строку изменения выравнивания "A".
Ascent
Свойство возвращает высоту максимального верхнего выносного элемента для строчных букв первой строки этого объекта MText.
Attachment
Свойство представляет тип точки присоединения этого объекта MText.
BackgroundFill
Свойство позволяет включить или отключить фоновую заливку для данного объекта MText.
BackgroundFillColor
Свойство представляет цвет фоновой заливки данного объекта MText.
BackgroundScaleFactor
Свойство представляет масштабный коэффициент для фона данного объекта MText.
BackgroundTransparency
Свойство представляет прозрачность фона данного объекта MText.
BlockBegin
Свойство представляет строку начала блока "{".
BlockEnd
Свойство представляет строку окончания блока "}".
ColorChange
Свойство возвращает строку изменения цвета "C".
ColumnAutoHeight
Свойство определяет, должны ли динамические столбцы использовать высоту MText или высоту, заданную пользователем для каждого столбца (каждой колонки текста).
ColumnCount
Свойство представляет количество столбцов (текстовых колонок) у данного объекта MText.
ColumnFlowReversed
Свойство определяет, является ли поток столбцов данного объекта MText обратным. Эта настройка влияет на способ добавления столбцов.
ColumnGutterWidth
Свойство представляет расстояние между столбцами текста (просвет между текстовыми колонками).
ColumnType
Свойство представляет тип текстовых колонок у данного объекта MText.
ColumnWidth
Свойство представляет ширину столбца, которая применяется к каждому столбцу данного объекта.
Contents
Свойство представляет копию символьного содержимого данного объекта MText. Коды форматирования, используемые для управления переносом, удаляются перед копированием строки.
Descent
Свойство возвращает высоту максимального нижнего выносного элемента букв последней строки этого объекта MText.
Direction
Свойство представляет вектор (WCS) направления оси X этого объекта MText.
FlowDirection
Свойство представляет направление потока данного многострочного текста.
FontChange
Свойство возвращает строку изменения шрифта "F".
Height
Свойство представляет высоту ограничивающего бокса данного объекта MText.
HeightChange
Свойство возвращает строку изменения высоты "H".
LineBreak
Свойство возвращает строку разрыва строки "p".
LineSpaceDistance
Свойство представляет межстрочный интервал в данном объекте.
LineSpacingFactor
Свойство представляет коэффициент межстрочного интервала в диапазоне [0,25 .. 4,0].
LineSpacingStyle
Свойство представляет стиль межстрочного интервала, который определяет применение этого интервала в зависимости от размера символов.
NonBreakSpace
Свойство возвращает строку неразрывного пробела "~".
Normal
Свойство представляет нормаль плоскости (WCS), в которой располагается данный объект MText.
ObliqueChange
Свойство возвращает строку изменения угла наклона текста "Q".
OverlineOff
Свойство возвращает строку перехода к незачёркнутому тексту "o".
OverlineOn
Свойство возвращает строку перехода к зачёркнутому тексту "O".
ParagraphBreak
Свойство возвращает строку перехода к новому абзацу "P".
Rotation
Свойство представляет угол поворота данного объекта MText (в радианах).
ShowBorders
Свойство управляет режимом отображения границ многострочного текста.
StackStart
Свойство возвращает строку перехода к дробному тексту "S".
Text
Свойство представляет текстовое содержимое многострочного текста, очищенное от кодов форматирования.
TextHeight
Свойство представляет номинальную высоту текста в данном объекте MText.
TextStyle
Свойство представляет идентификатор стиля текста, используемого данным объектом MText. Устарело. Рекомендуется использовать свойство TextStyleId.
TextStyleId
Свойство представляет идентификатор объекта стиля текста, используемого данным объектом MText.
TextStyleName
Свойство представляет имя текстового стиля, используемого данным объектом MText.
TrackChange
Свойство возвращает строку изменения трекинга "T", т.е. переход к другому межбуквенному расстоянию.
UnderlineOff
Свойство возвращает строку отмены подчеркивания текста "l".
UnderlineOn
Свойство возвращает строку перехода к подчеркнутому тексту "L".
UseBackgroundColor
Свойство определяет, используется ли цвет фона экрана в качестве цвета фона этого объекта MText.
Width
Свойство представляет ширину абзаца многострочного текста.
WidthChange
Свойство возвращает строку перехода к другой ширине текста "W".
location
Свойство представляет точку вставки данного объекта MText.
