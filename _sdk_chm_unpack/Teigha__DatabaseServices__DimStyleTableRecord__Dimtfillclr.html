Свойство отвечает за цвет заливки фона размерного текста при Dimtfill = 2 ("By DIMTFILLCLR").

C#
Visual Basic

public virtual Color Dimtfillclr { get; set; }

Копировать Код
Public Overridable Property Dimtfillclr As Color

Возвращает

 

Возвращает тип данных Teigha.Colors.Color.

Описание

 

В свойстве хранится цвет заливки фона размерного текста в виде типа данных Teigha.Colors.Color. Значение свойства учитывается, когда свойство Dimtfill равно 2 («By DIMTFILLCLR»).

В таблице приведены стандартные цвета. При необходимости, можно создать цвет, не включенный в таблицу, используя RGB-модель и изменить цвет заливки текста на этот цвет. Значение Dimtfillclr по умолчанию – «По блоку». 

Свойство Dimtfillclr также доступно через системную переменную DIMTFILLCLR и пользовательское меню в окне изменения/создания размерного стиля: 

 

При изменении системной переменной DIMTFILLCLR будет создан новый размерный стиль под названием <переопределение> на основе текущего с новым значением DIMTFILLCLR.

Пример

 

Пример изменения цвета заливки фона размерного текста (значения свойства Dimtfillclr) в размерном стиле Standard:

C#
Копировать Код
public partial class Commands { [Teigha.Runtime.CommandMethod("DimtfillclrChange")] public void DimtfillclrStandardDimStyle() { // Получение ссылки на активный документ HostMgd.ApplicationServices.Document doc = HostMgd.ApplicationServices.Application.DocumentManager.MdiActiveDocument; // Получение ссылки на редактор документа HostMgd.EditorInput.Editor ed = doc.Editor; // Получение ссылки на базу данных документа Teigha.DatabaseServices.Database db = doc.Database; // Начало транзакции с базой данных документа using (Teigha.DatabaseServices.Transaction trans = db.TransactionManager.StartTransaction()) { // Открываем таблицу размерных стилей для чтения Teigha.DatabaseServices.DimStyleTable dimStyleTable = trans.GetObject(db.DimStyleTableId, Teigha.DatabaseServices.OpenMode.ForRead) as Teigha.DatabaseServices.DimStyleTable; // Находим в таблице размерных стилей ObjectId нужного стиля по его названию Teigha.DatabaseServices.ObjectId styleId = dimStyleTable["Standard"]; // Открываем таблицу размерного стиля для изменений Teigha.DatabaseServices.DimStyleTableRecord obj = trans.GetObject(styleId, Teigha.DatabaseServices.OpenMode.ForWrite) as Teigha.DatabaseServices.DimStyleTableRecord; ed.WriteMessage($"nDimtfillclr before: {obj.Dimtfillclr}"); // Изменяем цвет заливки размерного текста на синий obj.Dimtfill = 2; obj.Dimtfillclr = Teigha.Colors.Color.FromColorIndex(Teigha.Colors.ColorMethod.ByColor, 5); ed.WriteMessage($"nDimtfillclr after: {obj.Dimtfillclr}"); trans.Commit(); } } }
