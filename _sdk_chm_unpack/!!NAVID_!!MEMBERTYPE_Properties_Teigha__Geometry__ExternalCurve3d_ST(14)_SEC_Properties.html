ExternalCurve3d Свойства  Условные обозначения
Свойства
Имя
Описание
ExternalCurve
Свойство ExternalCurve представляет внешнюю кривую в ее исходном виде.
ExternalCurveKind
Свойство ExternalCurveKind представляет информацию о нативной системе, в которой было создано определение внешней кривой.
IsCircularArc
Свойство IsCircularArc позволяет проверить, представляет ли данный объект круглую дугу.
IsDefined
Свойство IsDefined позволяет проверить, является ли данный объект определенным, т.е. сформировано ли определение данной кривой.
IsEllipticalArc
Свойство IsEllipticalArc позволяет проверить, представляет ли данный объект эллиптическую дугу.
IsLine
Свойство IsLine позволяет проверить, представляет ли данный объект прямую линию.
IsLineSegment
Свойство IsLineSegment позволяет проверить, представляет ли данный объект прямолинейный сегмент.
IsNativeCurve
Свойство IsNativeCurve позволяет проверить, поддерживает ли геометрическая библиотека платформы нативное представление данной внешней кривой.
IsNurbCurve
Свойство IsNurbCurve позволяет проверить, может ли экземпляр внешней кривой быть представлен определением, которое является нативным для графической системы платформы.
IsOwnerOfCurve
Свойство IsOwnerOfCurve позволяет убедиться, что данный объект является владельцем данных своего определения.
IsRay
Свойство IsRay позволяет проверить, представляет ли данный объект луч.
NativeCurve
Свойство NativeCurve обеспечивает доступ к нативному представлению данной внешней кривой в геометрической библиотеке платформы.
