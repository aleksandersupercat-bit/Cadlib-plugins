MTextFragment Свойства Свойства
MTextFragment Свойства  Условные обозначения
Свойства
Имя
Описание
BigFont
Свойство возвращает имя большого шрифта SHX.
Bold
Свойство позволяет проверить, представлен ли текст полужирным шрифтом.
CapsHeight
Свойство возвращает высоту текста.
Color
Свойство возвращает цвет текста.
Direction
Свойство возвращает направление оси X текста.
Extents
Свойство возвращает ограничивающий (габаритный) бокс данного объекта в системе координат объекта (OCS).
Italic
Свойство позволяет проверить, представлен ли текст наклонным шрифтом.
LineBreak
Свойство позволяет проверить, завершается ли данный фрагмент символом разрыва строки.
Location
Свойство возвращает точку вставки текста.
NewParagraph
Свойство позволяет проверить, является ли данный фрагмент началом нового абзаца текста.
Normal
Свойство представляет вектор нормали плоскости, в которой располагается данный объект.
ObliqueAngle
Свойство возвращает угол наклона текста.
Overlined
Свойство позволяет проверить, является ли текст надчёркнутым.
ShxFont
Свойство возвращает имя шрифта SHX.
StackBottom
Свойство позволяет проверить, представляет ли данный фрагмент знаменатель дробного текста.
StackTop
Свойство позволяет проверить, представляет ли данный фрагмент числитель дробного текста.
Text
Свойство возвращает текст данного фрагмента в виде строки, очищенной от кодов форматирования.
TrackingFactor
Свойство возвращает коэффициент трекинга в данном фрагменте.
TrueTypeFont
Свойство представляет имя шрифта TrueType.
Underlined
Свойство позволяет проверить, является ли текст фрагмента подчёркнутым.
WidthFactor
Свойство представляет коэффициент растяжения символов текста.
