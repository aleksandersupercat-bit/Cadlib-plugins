LoftOptionsBuilder Свойства
Послать отзыв

Ниже перечислены свойства класса LoftOptionsBuilder.

Свойства
Имя
Описание
AlignDirection
Свойство позволяет проверить состояние флага-индикатора "правильной" взаимной ориентации нормалей заданных поперечных сечений.
ArcLengthParam
Свойство позволяет проверить, применяется ли параметризация по длине дуги.
Closed
Свойство позволяет проверить состояние флага замкнутости получаемой поверхности.
DraftEnd
Свойство возвращает угол в радианах, под которым результирующая поверхность входит в последнее поперечное сечение (end draft angle, угол входа поверхности в сечение).
DraftEndMag
Свойство возвращает величину вытягивания для поверхности у последнего поперечного сечения (end draft magnitude).
DraftStart
Свойство возвращает угол в радианах, под которым результирующая поверхность вытягивается из первого поперечного сечения (start draft angle, угол выхода поверхности из сечения).
DraftStartMag
Свойство возвращает величину вытягивания для поверхности у первого поперечного сечения (start draft magnitude).
NoTwist
Свойство возвращает значение опции noTwist (нет кручения).
NormalOption
Свойство возвращает настройку для нормали к плоскости в данном объекте из перечисления LoftOptionsNormalOption.
Ruled
Свойство возвращает состояние флага ruled (кусочно-линейная поверхность).
Simplify
Свойство возвращает состояние флага simplify (упрощение), который определяет возможность использования простых типов поверхностей вместо общего там, где это возможно.
VirtualGuide
Свойство позволяет определить, используется ли при создании лофт-поверхностей опция virtual guide (виртуальная направляющая).
