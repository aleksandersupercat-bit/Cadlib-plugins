Перечисления
Имя
Описание
DefaultLightingType
Данное перечисление определяет количество используемых по умолчанию источников освещения.
InvalidationHint
Данное перечисление определяет типы данных, которые, будучи в кэше, могут потерять актуальность и требовать перестроения. Используется графической системой для управления методом GraphicsSystem.Model.Invalidate().
Projection
Перечисление Projection определяет типы проецирования, используемого камерой.
RenderMode
Перечисление RenderMode определяет режимы рендеринга вида.
RenderType
Перечисление RenderType определяет типы рендеринга геометрии модели, поддерживаемые графической системой.
SelectionMode
Данное перечисление определяет режимы выделения объектов.
SelectionReactorResult
Перечисление SelectionReactorResult определяет возможные результаты, которые предоставляют объекты SelectionReactor в рамках уведомлений о событиях выбора объектов.
ViewportFlags
Перечисление ViewportFlags определяет флаги видового экрана на устройстве.
