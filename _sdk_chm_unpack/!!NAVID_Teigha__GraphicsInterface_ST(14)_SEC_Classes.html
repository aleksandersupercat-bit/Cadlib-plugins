AbstractClipBoundary
Класс AbstractClipBoundary является базовым для классов, используемых для передачи дополнительной информации о границах обрезки из объектов Drawable в графическую систему платформы.
ClipBoundary
Класс ClipBoundary используется для передачи описаний границ обрезки отображаемых объектов в базовую систему.
ColorRGB
Структура ColorRGB предназначена для представления цвета в формате RGB.
CommonDraw
Класс CommonDraw абстрагирует общность классов ViewportDraw и WorldDraw. CommonDraw позволяет разрабатывать код графических процедур, который будет работать в обоих случаях.
Context
Класс Context определяет общие операции и свойства, используемые платформой в процессе векторизации.
ContextForDbDatabase
Класс ContextForDbDatabase представляет операции и свойства, используемые для векторизации объектов базы данных.
DisplayStyle
Класс DisplayStyle представляет стиль отображения как набор свойств, используемых графической системой при отображении графических объектов.
DistantLightTraits
Класс DistantLightTraits представляет характеристики удаленного источника света.
Drawable
Класс представляет графические объекты, как временные, так и постоянные, которые реализуют интерфейс графического отображения.
DrawableOverrule
Класс предназначен для создания переопределения графического представления стандартных примитивов nanoCAD.
DrawableTraits
Класс DrawableTraits реализует интерфейс к графическим атрибутам графических примитивов.
EdgeData
Класс EdgeData представляет массивы атрибутов ребер сетей и оболочек.
EdgeStyle
Класс EdgeStyle определяет набор свойств визуального стиля для отображения ребер графических объектов.
FaceData
Класс FaceData служит контейнером для организации опциональных данных, используемых для множества граней графического примитива сети или оболочки.
FaceStyle
Класс FaceStyle определяет визуальный стиль как набор свойств, применяемых к отображаемым граням графических объектов.
FontDescriptor
Структура FontDescriptor представляет ряд характеристик шрифта, такие как гарнитура, набор символов, пропорции и семейство, жирный или курсивный шрифт.
FrontAndBackClipping
Структура FrontAndBackClipping представляет данные, которые используются при обрезке видовых экранов.
GIRasterImage
Класс GIRasterImage представляет абстракцию растрового изображения в графической подсистеме платформы.
GIRasterImageDesc
Класс GIRasterImageDesc представляет фиктивную реализацию интерфейса GIRasterImage.
GdiDrawObject
Класс GdiDrawObject реализует интерфейс, который позволяет объекту-владельцу GDI использовать новый OwnerDraw().
Geometry
Класс Geometry абстрагирует общее между классами ViewportGeometry и WorldGeometry.
GradientBackgroundTraits
Класс GradientBackgroundTraits представляет характеристики градиентного фона базовой системы отображения.
GroundPlaneBackgroundTraits
Класс GroundPlaneBackgroundTraits представляет характеристики фона нулевой плоскости.
ImageBGRA32
Класс ImageBGRA32 представляет изображение с параметрами цвета в формате RGBA (красный, зелёный, синий, альфа-канал).
ImageBackgroundTraits
Класс ImageBackgroundTraits представляет характеристики фонов на основе изображений.
LightAttenuation
Класс LightAttenuation представляет параметры затухания излучения источника света.
LightTraits
Класс LightTraits представляет состояние источника излучения света (вкл./выкл.).
LinetypeCollection
Класс LinetypeCollection представляет коллекцию типов линий.
Mapper
Класс Mapper управляет наложением текстуры (карты) материала на поверхность объектов.
MaterialColor
Класс MaterialColor представляет цвета материалов.
MaterialDiffuseComponent
Класс MaterialDiffuseComponent представляет диффузный компонент материала.
MaterialMap
Класс MaterialMap реализует карты материалов.
MaterialNormalMapComponent
Структура MaterialNormalMapComponent представляет карту материала для канала нормального компонента наложения материала, используемый метод вычисления и вклад (strength) этого канала.
MaterialOpacityComponent
Структура MaterialOpacityComponent представляет компонент непрозрачности для материала.
MaterialRefractionComponent
Структура MaterialRefractionComponent представляет компонент преломления материала.
MaterialSpecularComponent
Структура MaterialSpecularComponent представляет компонент зеркального отражения материала (блики).
MaterialTexture
Класс MaterialTexture является абстрактным базовым классом для всех текстур материалов.
MentalRayRenderSettingsTraits
Класс MentalRayRenderSettingsTraits представляет настройки рендеринга для движка Mental Ray.
MentalRayRenderSettingsTraits2
Класс MentalRayRenderSettingsTraits2 представляет настройки рендеринга для движка Mental Ray.
MentalRayRenderSettingsTraitsBoolParameter
Структура MentalRayRenderSettingsTraitsBoolParameter представляет параметры, которые определяют для рендерера Mental Ray флаги использования пользовательских радиусов выборки в режиме финальной сборки.
MentalRayRenderSettingsTraitsDiagnosticGridModeParameter
Структура MentalRayRenderSettingsTraitsDiagnosticGridModeParameter представляет параметры, используемые для задания режима диагностической сетки и размера этой сетки.
MentalRayRenderSettingsTraitsDoubleRangeParameter
Структура MentalRayRenderSettingsTraitsDoubleRangeParameter представляет определяемые пользователем радиусы выборки в режиме финальной сборки рендерера Mental Ray.
MentalRayRenderSettingsTraitsFloatParameter
Структура MentalRayRenderSettingsTraitsFloatParameter представляет параметры, используемые для указания контрастного цвета выборки для рендерера Mental Ray.
MentalRayRenderSettingsTraitsIntegerRangeParameter
Структура MentalRayRenderSettingsTraitsIntegerRangeParameter представляет параметры, которые определяют дискретизацию выборок, используемых в процессе затенения пикселей при рендеринге.
MentalRayRenderSettingsTraitsSamplingParameter
Структура MentalRayRenderSettingsTraitsSamplingParameter представляет параметры фильтра, который используется для формирования цвета пикселя на основе нескольких выборок.
MentalRayRenderSettingsTraitsTraceParameter
Структура MentalRayRenderSettingsTraitsTraceParameter представляет максимальную глубину (уровень рекурсии) при трассировке лучей или фотонов.
NonEntityTraits
Класс NonEntityTraits реализует пустые интерфейсы, которые позволяют предоставить доступ к уникальным графическим атрибутам отображаемых негеометрических объектов через интерфейс DrawableTraits.
PathNode
Класс PathNode представляет узел пути отображаемого графической системой объекта.
PixelBGRA32
Структура PixelBGRA32 определяет цвет пикселя в формате RGBA, который представляет цвета, используя целочисленные значения красной, зеленой и синей компонент, а также альфа-канала, управляющего непрозрачностью.
PixelFormatInfo
Структура PixelFormatInfo представляет данные пиксельного формата.
Polyline
Класс Polyline реализует эквивалент облегченной полилинии, которая используется для передачи геометрии полилиний в подсистему векторизации платформы nanoCAD.
PolylineCollection
Класс PolylineCollection представляет коллекцию облегченных полилиний GraphicsInterface.Polyline.
RenderSettingsTraits
Класс RenderSettingsTraits представляет настройки рендеринга для базовой системы отображения.
ShadowParameters
Класс ShadowParameters представляет связанные с тенью свойства источников света.
StandardLightTraits
Класс StandardLightTraits представляет характеристики стандартного источника света.
SubEntityTraits
Этот класс используется для управления текущими значениями атрибутов цвета, слоя, типа линии, типа заливки и атрибутов системного маркера для графических примитивов.
TextStyle
Класс TextStyle предоставляет интерфейс для настройки атрибутов отображения текста.
TransientManager
Класс TransientManager представляет функционал менеджера временных отображаемых объектов растровой графики.
Variant
Класс Variant служит основой для универсального хранилища данных, используемого для определений процедурных материалов.
VertexData
Класс VertexData представляет контейнер для массивов атрибутов вершин сетей и оболочек.
Viewport
Этот класс представляет характеристики текущего видового экрана, используемые механизмами векторизации графического интерфейса. Эти характеристики представляют интерес для оптимизации графического представления пользовательских объектов с учетом конкретной конфигурации видового экрана.
ViewportDraw
Класс предназначен для получения характеристик текущего видового экрана.
ViewportGeometry
Данный класс определяет методы, которые позволяют объектам себя векторизовать.
ViewportTraits
Класс ViewportTraits предоставляет видовым экранам возможность передавать специфические свойства конкретного видового экрана в системы отображения и рендеринга.
VisualStyle
Класс VisualStyle представляет набор визуальных свойств, которые можно применять к графическим объектам.
VisualStyleTraits
Класс VisualStyleTraits представляет атрибуты визуального стиля.
WorldDraw
Класс WorldDraw представляет функциональность независимой от видового экрана векторизации на уровне графических объектов.
WorldGeometry
Класс WorldGeometry отвечает за помещение геометрии в графический кэш для последующего отображения. Объект этого класса является частью каждого объекта WorldDraw, передаваемого в метод WorldDraw() объекта, которому потребовалась генерация графики.
