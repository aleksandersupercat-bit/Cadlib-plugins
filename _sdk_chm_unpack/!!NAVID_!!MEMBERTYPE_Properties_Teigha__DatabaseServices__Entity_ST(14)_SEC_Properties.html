Entity Свойства Свойства
Entity Свойства  Условные обозначения
Свойства
Имя
Описание
BlockId
Свойство предназначено для получения идентификатора владельца геометрического объекта, которым должен быть объект типа BlockTableRecord.
BlockName
Свойство предназначено для получения названия владельца геометрического объекта, которым должен быть объект типа BlockTableRecord.
CastShadows
Свойство предназначено для получения или установки флага, определяющего, будет ли геометрический объект участвовать в отбрасывании тени при визуализации сцены.
CloneMeForDragging
Свойство предназначено для определения, разрешено ли создание клона геометрического объекта для использования его в операции перетаскивания.
CollisionType
Свойство предназначено для получения типа взаимодействия геометрического объекта при обнаружении коллизий, что определяет его поведение в операциях, учитывающих пространственные пересечения.
Color
Свойство предназначено для установки или получения полной информации о цвете геометрического объекта с использованием экземпляра класса Teigha.Colors.Color.
ColorIndex
Свойство предназначено для получения или установки номера цвета, назначенного геометрическому объекту в соответствии с текущей палитрой цветов чертежа.
CompoundObjectTransform
Свойство предназначено для получения матрицы преобразования, применяемой к внутренним объектам составного геометрического объекта.
Ecs
Свойство предназначено для получения матрицы преобразования, которая позволяет перевести точки или векторы из локальной системы координат объекта (OCS) в мировую систему координат (WCS).
EdgeStyleId
Свойство предназначено для получения или установки идентификатора визуального стиля, применяемого исключительно к ребрам геометрического объекта.
EntityColor
Свойство предназначено для установки или получения цветовых настроек геометрического объекта в виде экземпляра класса EntityColor, который может содержать информацию о цвете по индексу, true-цвете или цвете слоя.
FaceStyleId
Свойство предназначено для получения или установки идентификатора визуального стиля, применяемого к поверхностям (граням) трехмерного геометрического объекта.
ForceAnnoAllVisible
Свойство предназначено для получения или установки значения системной переменной ANNOALLVISIBLE, которая определяет, будут ли все аннотативные объекты видны при различных масштабах представления.
GeometricExtents
Свойство предназначено для получения ограничивающего параллелепипеда, содержащего трехмерное представление геометрического объекта, с угловыми точками, выраженными в мировой системе координат (WCS).
Hyperlinks
Свойство предоставляет доступ к коллекции гиперссылок, связанных с текущим геометрическим объектом.
IsPlanar
Свойство предназначено для определения, лежит ли геометрический объект полностью в одной плоскости.
Layer
Свойство предназначено для получения или установки имени слоя (записи таблицы слоёв LayerTableRecord), с которым связан данный геометрический объект.
LayerId
Свойство предназначено для получения или установки идентификатора записи таблицы слоёв (LayerTableRecord), которая связана с текущим геометрическим объектом.
LineWeight
Свойство предназначено для получения или установки толщины линии (ширины) геометрического объекта, которая определяет её отображение на экране и при печати.
Linetype
Свойство предназначено для получения или установки имени типа линии (записи таблицы типов линий LinetypeTableRecord), которая связана с данным геометрическим объектом.
LinetypeId
Свойство предназначено для получения или установки идентификатора записи таблицы типов линий (LinetypeTableRecord), которая связана с текущим геометрическим объектом.
LinetypeScale
Свойство предназначено для получения или установки масштаба типа линии геометрического объекта, который определяет, как будет отображаться повторяющийся узор линии (например, пунктир) на чертеже.
Material
Свойство предназначено для получения или установки имени материала, назначенного геометрическому объекту, что влияет на его внешний вид при визуализации.
MaterialId
Свойство предназначено для получения или установки идентификатора материала, связанного с геометрическим объектом, что определяет его визуальные свойства при трёхмерной визуализации.
MaterialMapper
Свойство предназначено для получения или установки объекта-маппера, который определяет, как текстура материала отображается на поверхности геометрического объекта.
PlotStyleName
Свойство предназначено для получения или установки имени стиля печати, применяемого к геометрическому объекту при выводе чертежа на печать.
PlotStyleNameId
Свойство предназначено для получения или установки идентификатора стиля печати (PlotStyleDescriptor), связанного с геометрическим объектом, что определяет его параметры вывода на печать.
ReceiveShadows
Свойство предназначено для получения или установки флага, определяющего, может ли данный геометрический объект принимать тени от других объектов при трёхмерной визуализации.
Transparency
Свойство предназначено для получения или установки уровня прозрачности геометрического объекта, что влияет на его визуальное отображение при трёхмерной визуализации.
Visible
Свойство предназначено для получения или установки состояния видимости геометрического объекта в чертеже.
VisualStyleId
Свойство предназначено для получения или установки идентификатора полного визуального стиля, применяемого к геометрическому объекту, что определяет его внешний вид при трёхмерной визуализации.
