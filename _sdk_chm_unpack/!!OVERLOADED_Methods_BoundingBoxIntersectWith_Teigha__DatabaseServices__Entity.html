Entity::BoundingBoxIntersectWith (Entity^, Intersect, Plane^, Point3dCollection^, IntPtr, IntPtr)
Метод предназначен для расчета проекции края ограничивающего бокса исходного объекта чертежа и объекта, пересечения с которым необходимо найти, на указанную плоскость проекции. После вычисления проекции метод определяет ограничивающий бокс для объекта, находит точки пересечения и проецирует их обратно на край ограничивающего прямоугольника исходного объекта.
Entity::BoundingBoxIntersectWith (Entity^, Intersect, Plane^, Point3dCollection^, int, int)
Метод предназначен для проецирования ребер ограничивающего параллелепипеда данного геометрического объекта и другого геометрического объекта на указанную плоскость проекции, поиска точек пересечений и последующего проецирования этих точек обратно на ребра ограничивающего параллелепипеда данного объекта. Все найденные точки добавляются в выходной массив и представляют собой точки ребер ограничивающего параллелепипеда данного геометрического объекта.
Entity::BoundingBoxIntersectWith (Entity^, Intersect, Point3dCollection^, IntPtr, IntPtr)
Метод предназначен для расчета ограничивающего параллелепипеда данного геометрического объекта в мировой системе координат (WCS) и поиска всех пересечений указанного геометрического объекта (entityPointer) с ребрами этого параллелепипеда. Найденные точки пересечений добавляются в выходной массив и задаются в координатах WCS.
Entity::BoundingBoxIntersectWith (Entity^, Intersect, Point3dCollection^, int, int)
Метод предназначен для расчета ограничивающего параллелепипеда данного геометрического объекта в мировой системе координат (WCS) и определения всех пересечений указанного геометрического объекта (entityPointer) с ребрами этого параллелепипеда.
