Ниже перечислены свойства класса MLeaderStyle.

Свойства
Имя
Описание
Annotative
Свойство представляет предусмотренное стилем состояние аннотаций.
ArrowSize
Свойство представляет размер стрелки, предусмотренный данным стилем.
ArrowSymbolId
Свойство представляет идентификатор объекта символа стрелки, предусмотренный данным стилем.
BlockColor
Свойство представляет цвет, предусмотренный данным стилем для отрисовки содержимого блока мультивыноски.
BlockConnectionType
Свойство представляет предусмотренный данным стилем вариант соединения блока с линиями выноски.
BlockId
Свойство представляет идентификатор ObjectId блока, связанного с данным стилем.
BlockRotation
Свойство представляет угол поворота блока, на который ссылается данный стиль.
BreakSize
Свойство представляет величину разрыва выносных линий, предусмотренную данным стилем.
ContentType
Свойство представляет тип содержимого мультивыноски, предусмотренный данным стилем.
DefaultMText
Свойство представляет текст по умолчанию для связанного многострочного текста.
DoglegLength
Свойство представляет длину полки мультивыноски, предусмотренную данным стилем.
DrawLeaderOrderType
Свойство представляет последовательность отрисовки сегментов выносных линий объектов MLeader, предусмотренный данным стилем.
DrawMLeaderOrderType
Свойство представляет порядок отрисовки содержимого объектов MLeader, предумотренный данным стилем.
EnableBlockRotation
Свойство позволяет проверить, предусмотрено ли данным стилем вращение связанного блока.
EnableBlockScale
Свойство позволяет включить или отключить масштабирование содержимого связанного блока.
EnableDogleg
Свойство позволяет контролировать отображение полки на выносной линии мультивыноски.
EnableFrameText
Свойство позволяет контролировать отображение рамки вокруг текста мультивыноски.
EnableLanding
Свойство должно управлять отображением зазора между концом линии полки и содержимым мультивыноски. В текущей реализации платформы всегда считается равным true.
FirstSegmentAngleConstraint
Свойство представляет угловое ограничение для первого сегмента линии выноски.
LandingGap
Свойство представляет величину зазора между концом линии полки и многострочным текстом мультивыноски.
LeaderLineColor
Свойство представляет цвет линии выноски, предусмотренный данным стилем.
LeaderLineType
Свойство представляет предусмотренный данным стилем тип линий выноски (прямая, сплайн, невидимая).
LeaderLineTypeId
Свойство представляет предусмотренный данным стилем идентификатор ObjectId типа линии, используемого для отрисовки выносных линий мультивыноски.
LeaderLineWeight
Свойство представляет предусмотренную данным стилем толщину (вес) выносных линий мультивыноски.
MaxLeaderSegmentsPoints
Свойство представляет максимальное количество точек сегментов, используемых для построения линий выносок.
Name
Свойство представляет имя данного стиля мультивыноски.
Scale
Свойство представляет масштабный множитель для данного стиля.
SecondSegmentAngleConstraint
Свойство представляет угловое ограничение для второго сегмента линии выноски.
TextAlignAlwaysLeft
Свойство позволяет указать, что создаваемый многострочный текст должен всегда выравниваться по левому краю.
TextAlignmentType
Свойство представляет тип выравнивания многострочного текста мультивыноски, предусмотренный данным стилем.
TextAngleType
Свойство представляет выбор угла для размещения многострочного текста относительно последнего сегмента линии выноски.
TextAttachmentDirection
Свойство представляет ориентацию полки выносных линий, предусмотренную данным стилем.
TextAttachmentType
Свойство представляет предусмотренный стилем вариант расположения точки прикрепления выносной линии к многострочному тексту.
TextColor
Свойство представляет цвет многострочного текста, предусмотренный данным стилем.
TextHeight
Свойство представляет высоту многострочного текста, которая предусмотрена данным стилем.
TextStyleId
Свойство представляет идентификатор ObjectId текстового стиля, который должен применяться к многострочному тексту мультивыноски.
blockScale
Свойство представляет масштаб отображения блока, на который ссылается данный стиль.
