Ниже перечислены методы класса Dimension.

Методы
Методы
Имя
Описание
FieldFromMText
Метод копирует текстовое поле из указанного объекта MText, если он содержит такое поле, и добавляет его копию к данному размеру.
FieldToMText
Метод копирует текстовое поле из данного размера, если он содержит такое поле, и добавляет его копию к указанному объекту MText.
FormatMeasurement
Метод возвращает строку MText, представляющую форматированное значение размера объекта Dimension.
GenerateLayout
Метод предназначен для обновления размера в соответствии с текущим размерным стилем. На данный момент не реализован.
GetDimstyleData
Метод предназначен для копирования настроек размерного стиля данного объекта Dimension, включая переопределения, в указанную запись таблицы размерных стилей.
RecomputeDimensionBlock
Метод предназначен для перевычисления размерного блока, на который ссылается данный объект Dimension, чтобы отразить любые изменения, внесенные в объект Dimension с момента последнего обновления записи таблицы блоков.
RemoveTextField
Метод предназначен для удаления текстового поля из этого объекта Dimension, если такое поле имеется.
SetDimstyleData
Метод предназначен для копирования настроек размерного стиля, включая все переопределения, из указанной записи таблицы размерных стилей в данный объект Dimension.
