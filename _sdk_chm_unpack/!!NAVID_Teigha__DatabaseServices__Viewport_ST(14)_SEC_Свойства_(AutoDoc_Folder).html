TwistAngle
Свойство TwistAngle представляет угол поворота данного видового экрана в системе координат дисплея (DCS).
UcsFollowModeOn
Свойство UcsFollowModeOn позволяет определить, включен ли для данного видового экрана переход к отображению в плане при переключении с одной пользовательской системы координат (ПСК, UCS) на другую.
UcsIconAtOrigin
Свойство UcsIconAtOrigin позволяет определить, располагается ли в данном видовом экране значок ПСК (UCS) в начале пользовательской системы координат.
UcsIconVisible
Свойство UcsIconVisible позволяет определить, включено ли отображение значка ПСК (UCS) в данном видовом экране.
UcsName
Свойство UcsName представляет идентификатор объекта пользовательской системы координат, связанной с данным видовым экраном.
UcsOrthographic
Свойство UcsOrthographic представляет ортогональный вид в пользовательской системе координат.
UcsPerViewport
Свойство UcsPerViewport позволяет определить, следует ли сохранять связь ПСК с данным видовым экраном.
ViewCenter
Свойство ViewCenter представляет точку центра вида в данном видовом экране в системе координат дисплея.
ViewDirection
Свойство ViewDirection представляет направление вида в данном видовом экране (в мировой системе координат).
ViewHeight
Свойство ViewHeight представляет высоту вида данного видового экрана в системе координат дисплея (DCS).
ViewOrthographic
Свойство ViewOrthographic представляет текущий ортогональный вид в данном видовом экране.
VisualStyleId
Свойство VisualStyleId представляет идентификатор объекта визуального стиля, который используется в данном видовом экране.
Width
Свойство Width представляет ширину данного видового экрана.
centerPoint
Свойство CenterPoint представляет центральную точку данного видового экрана в координатах МСК (WCS).
viewTarget
Свойство ViewTarget представляет целевую точку данного видового экрана в мировой системе координат.
