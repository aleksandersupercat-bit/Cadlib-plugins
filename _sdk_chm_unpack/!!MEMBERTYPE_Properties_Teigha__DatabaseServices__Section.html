Ниже перечислены свойства класса Section.

Свойства
Имя
Описание
BottomPlane
Свойство BottomPlane представляет высоту индикатора секущей плоскости ниже линии сечения.
Boundary
Свойство Boundary представляет секущую плоскость, ограниченную линией сечения, двумя боковыми линиями и задней линией.
Elevation
Свойство Elevation представляет уровень плоскости по оси Z.
IndicatorFillColor
Свойство IndicatorFillColor представляет цвет заливки индикатора секущей плоскости.
IndicatorTransparency
Свойство IndicatorTransparency представляет прозрачность индикатора секущей плоскости в интервале [1–100].
IsLiveSectionEnabled
Свойство IsLiveSectionEnabled позволяет определить, включен ли режим псевдоразреза.
Name
Свойство Name представляет имя секущей плоскости.
Normal
Свойство Normal представляет вектор нормали секущей плоскости.
NumVertices
Свойство NumVertices представляет количество вершин на секущей плоскости.
Settings
Свойство Settings представляет настройки этого сечения или разреза.
State
Свойство State представляет тип секущей плоскости.
TopPlane
Свойство TopPlane представляет высоту индикатора секущей плоскости выше линии сечения.
VerticalDirection
Свойство VerticalDirection представляет вектор вертикального направления этой секущей плоскости.
Vertices
Свойство Vertices представляет коллекцию вершин данной секущей плоскости.
ViewingDirection
Свойство ViewingDirection представляет вектор направления обзора этой секущей плоскости.
