Public ReadOnly Property Options As PromptEntityOptions

Возвращает

Возвращает объект типа PromptEntityOptions, содержащий параметры выбора объекта. Если объект не был инициализирован, может вернуть null.

Описание

Свойство предоставляет доступ к объекту PromptEntityOptions, содержащему настройки команды выбора объекта в nanoCAD. Оно используется в обработчике события PromptingForEntity для динамической модификации параметров ввода перед началом операции — например, при вызове метода GetEntity(). 

Через это свойство можно:

  • задать текст приглашения (Message),
  • указать допустимые типы объектов с помощью методов SetAllow... (например, SetAllowLines, SetAllowCircles),
  • добавить ключевые слова,
  • разрешить или запретить пропуск ввода (AllowNone),
  • включить произвольный ввод (AllowArbitraryInput).

Свойство доступно только для чтения, но сам объект PromptEntityOptions может быть изменён, что позволяет гибко настраивать поведение команды в реальном времени.
