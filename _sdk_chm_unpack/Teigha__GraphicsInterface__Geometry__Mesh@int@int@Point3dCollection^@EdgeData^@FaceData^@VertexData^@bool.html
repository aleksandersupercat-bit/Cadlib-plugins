Описание

Метод Mesh предназначен для создания и отображения примитива сети, характеристики которой определяются объектами, переданными в метод в качестве параметров. 

Аргумент edgeData указывает на объект EdgeData, который содержит массивы цвета, слоя, видимости, типа линии и настройки маркера выделения ребер сети. Каждый из массивов в указанном объекте EdgeData должен иметь значение NULL или содержать (row - 1) x column + row x (column -1) элементов. 

Аргумент faceData указывает на объект FaceData, который содержит массивы цвета, слоя, видимости, типа линии и настройки маркера выделения граней. Каждый из массивов в указанном объекте FaceData должен иметь значение NULL или содержать (row -1) x (column - 1) элементов. 

Аргумент vertexData указывает на существующий объект VertexData, который содержит флаг ориентации вершин и указатель на массив нормалей в вершинах. Массив нормалей в вершинах сети должен быть либо NULL, либо содержать (row x column) элементов. 

Следует отметить, что метод не изменяет и не удаляет массив параметров или объекты. За используемую ими память отвечает вызывающая программа.
