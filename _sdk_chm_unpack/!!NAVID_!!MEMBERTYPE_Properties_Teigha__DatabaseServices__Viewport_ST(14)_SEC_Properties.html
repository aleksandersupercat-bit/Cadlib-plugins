Viewport Свойства Свойства
Viewport Свойства  Условные обозначения
Свойства
Имя
Описание
AmbientLightColor
Свойство AmbientLightColor представляет цвет рассеянного света в данном видовом экране.
AnnotationScale
Свойство AnnotationScale представляет объект масштаба аннотаций в данном видовом экране.
BackClipDistance
Свойство BackClipDistance представляет расстояние до задней плоскости отсечения для данного видового экрана.
BackClipOn
Свойство BackClipOn позволяет определить, включен ли в данном видовом экране режим заднего отсечения.
Background
Свойство Background представляет идентификатор объекта фона в данном видовом экране.
Brightness
Свойство Brightness представляет значение яркости, которое влияет на относительную интенсивность освещения в данном видовом экране.
CircleSides
Свойство CircleSides представляет процент масштабирования окружностей в данном видовом экране в диапазоне [1..20000]. Это значение используется при тесселяции кривых. Чем оно выше, тем больше линейных сегментов используется при тесселяции кривой.
Contrast
Свойство Contrast представляет значение контрастности, влияющее на интенсивность рассеянного освещения от других источников света в данном видовом экране.
CustomScale
Свойство CustomScale представляет пользовательский масштаб в данном видовом экране.
DefaultLightingOn
Свойство DefaultLightingOn позволяет определить, включено ли в данном видовом экране освещение по умолчанию.
DefaultLightingType
Свойство DefaultLightingType представляет тип освещения по умолчанию в данном видовом экране.
EffectivePlotStyleSheet
Свойство EffectivePlotStyleSheet представляет имя таблицы стилей печати, связанной с данным видовым экраном.
Elevation
Свойство Elevation представляет возвышение плоскости ПСК (UCS) данного видового экрана.
FastZoomOn
Свойство FastZoomOn позволяет определить, включен ли в данном видовом экране режим быстрого масштабирования.
FrontClipAtEyeOn
Свойство FrontClipAtEyeOn позволяет определить, проходит ли передняя плоскость отсечения через камеру.
FrontClipDistance
Свойство FrontClipDistance представляет расстояние до передней плоскости отсечения в данном видовом экране.
FrontClipOn
Свойство FrontClipOn позволяет определить, включен ли в данном видовом экране режим переднего отсечения.
GridAdaptive
Свойство GridAdaptive позволяет определить, является ли сетка в данном видовом экране адаптивной.
GridBoundToLimits
Свойство GridBoundToLimits определяет, отображается ли сетка в этом видовом экране только в границах лимитов МСК (WCS) и ПСК (UCS).
GridFollow
Свойство GridFollow определяет, реагирует ли сетка на динамические изменения ПСК (UCS) в этом видовом экране.
GridIncrement
Свойство GridIncrement представляет шаг сетки вдоль осей X и Y ПСК в данном видовом экране.
GridMajor
Свойство GridMajor представляет количество вспомогательных линий сетки между каждой парой соседних основных линий сетки.
GridOn
Свойство GridOn позволяет определить, включен ли режим отображения сетки в данном видовом экране.
GridSubdivisionRestricted
Свойство GridSubdivisionRestricted позволяет определить, запрещено ли дробление сетки мельче шага вспомогательных линий сетки в данном видовом экране.
Height
Свойство Height представляет высоту этого видового экрана в единицах чертежа.
HiddenLinesRemoved
Свойство HiddenLinesRemoved позволяет определить, будут ли удалены скрытые линии в данном видовом экране при выводе на печать.
LensLength
Свойство LensLength представляет фокусное расстояние в режиме вида в перспективе (в мм) в данном видовом экране.
LinkedToSheetView
Свойство LinkedToSheetView позволяет определить, связан ли данный видовой экран с видом в пространстве листа.
Locked
Свойство Locked позволяет определить, заблокирован ли масштабный коэффициент в данном видовом экране.
NonRectClipEntityId
Свойство NonRectClipEntityId представляет идентификатор объекта чертежа, который послужил инструментом обрезки в данном видовом экране.
NonRectClipOn
Свойство NonRectClipOn позволяет определить, включен ли в данном видовом экране режим обрезки непрямоугольным объектом.
Number
Свойство Number представляет идентификационный номер данного видового экрана.
On
Свойство On позволяет определить состояние данного видового экрана (включен или выключен).
PerspectiveOn
Свойство PerspectiveOn позволяет определить, включено ли перспективное проецирование в данном видовом экране.
PlotAsRaster
Свойство PlotAsRaster позволяет определить, выводится ли данный видовой экран на печать в растровом режиме с использованием своих текущих настроек тонирования.
PlotStyleSheet
Свойство PlotStyleSheet представляет имя таблицы стилей печати, применяемых к объектам в данном видовом экране.
PlotWireframe
Свойство PlotWireframe позволяет определить, отображается ли векторная графика в данном видовом экране с использованием текущих настроек отображения теней при выводе на печать.
RenderMode
Свойство RenderMode представляет текущий режим рендеринга в данном видовом экране.
ShadePlot
Свойство ShadePlot представляет текущий режим отображения теней в данном видовом экране.
ShadePlotId
Свойство ShadePlotId представляет объект тени (визуальный стиль или предопределенные настройки рендеринга), который определяет, как будет выводиться на печать текущий видовой экран.
SnapAngle
Свойство SnapAngle представляет угол поворота шаговой привязки в данном видовом экране относительно текущей ПСК (UCS).
SnapBasePoint
Свойство SnapBasePoint представляет базовую точку шаговой привязки в данном видовом экране относительно текущей ПСК (UCS).
SnapIncrement
Свойство SnapIncrement представляет интервал шаговой привязки в данном видовом экране.
SnapIsoPair
Свойство SnapIsoPair представляет плоскость изометрии для данного видового экрана.
SnapIsometric
Свойство SnapIsometric позволяет определить, является ли изометрическая привязка текущим типом шаговой привязки.
SnapOn
Свойство SnapOn позволяет определить, включен ли режим шаговой привязки в данном видовом экране.
StandardScale
Свойство StandardScale представляет масштаб в данном видовом экране, используя стандартные значения масштабов из перечисления Teigha.DatabaseServices.StandardScaleType.
SunId
Свойство SunId представляет идентификатор связанного с этим видовым экраном объекта солнца (бесконечно удаленного источника освещения).
Thumbnail
Свойство Thumbnail представляет изображение предварительного просмотра этого видового экрана.
Transparent
Свойство Transparent позволяет определить, выставлен ли флаг прозрачности для данного видового экрана.
TwistAngle
Свойство TwistAngle представляет угол поворота данного видового экрана в системе координат дисплея (DCS).
UcsFollowModeOn
Свойство UcsFollowModeOn позволяет определить, включен ли для данного видового экрана переход к отображению в плане при переключении с одной пользовательской системы координат (ПСК, UCS) на другую.
UcsIconAtOrigin
Свойство UcsIconAtOrigin позволяет определить, располагается ли в данном видовом экране значок ПСК (UCS) в начале пользовательской системы координат.
UcsIconVisible
Свойство UcsIconVisible позволяет определить, включено ли отображение значка ПСК (UCS) в данном видовом экране.
UcsName
Свойство UcsName представляет идентификатор объекта пользовательской системы координат, связанной с данным видовым экраном.
UcsOrthographic
Свойство UcsOrthographic представляет ортогональный вид в пользовательской системе координат.
UcsPerViewport
Свойство UcsPerViewport позволяет определить, следует ли сохранять связь ПСК с данным видовым экраном.
ViewCenter
Свойство ViewCenter представляет точку центра вида в данном видовом экране в системе координат дисплея.
ViewDirection
Свойство ViewDirection представляет направление вида в данном видовом экране (в мировой системе координат).
ViewHeight
Свойство ViewHeight представляет высоту вида данного видового экрана в системе координат дисплея (DCS).
ViewOrthographic
Свойство ViewOrthographic представляет текущий ортогональный вид в данном видовом экране.
VisualStyleId
Свойство VisualStyleId представляет идентификатор объекта визуального стиля, который используется в данном видовом экране.
Width
Свойство Width представляет ширину данного видового экрана.
centerPoint
Свойство CenterPoint представляет центральную точку данного видового экрана в координатах МСК (WCS).
viewTarget
Свойство ViewTarget представляет целевую точку данного видового экрана в мировой системе координат.
