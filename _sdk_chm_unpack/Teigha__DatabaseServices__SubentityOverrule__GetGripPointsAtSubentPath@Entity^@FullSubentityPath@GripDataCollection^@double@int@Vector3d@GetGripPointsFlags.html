Описание

Данный метод переопределяет метод Entity.GetGripPointsAtSubentityPath() и вызывает его при реализации по умолчанию. Метод Entity.GetGripPointsAtSubentityPath() в реализации по умолчанию возвращает коллекцию ручек интересующего подобъекта. С помощью метода GetGripPointsAtSubentPath() настоящего класса можно переопределить механизм работы этого метода.
