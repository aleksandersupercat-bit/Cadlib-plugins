Параметры
Параметры
Описание
feature
Имя приложения или функции, создавших запись. Тип String.
modifiedOnly
Флаг, который определяет, следует ли возвращать только измененные записи. Тип bool.
affectsGraphicsOnly
Флаг, который определяет, следует ли возвращать только те записи, которые влияют на отображаемый вид. Тип bool.
walkXRefTree
Флаг, который определяет, следует ли просматривать списки файловых зависимостей во внешних ссылках. Тип bool.
Описание

Метод инициализирует итератор для раздела .dwg списка файловых зависимостей.

Замечания

Если аргументы не указаны, просматриваются все записи в локальном списке. Первые три предоставленных аргумента ограничивают список просматриваемых записей.
