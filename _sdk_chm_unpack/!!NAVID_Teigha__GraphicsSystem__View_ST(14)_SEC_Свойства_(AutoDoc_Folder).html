Teigha.GraphicsSystem.View
Имя
Описание
BackClip
Свойство BackClip представляет удаление задней плоскости отсечения вида от точки цели камеры вдоль направления взгляда камеры.
Background
Свойство Background представляет идентификатор объекта, который является производным от класса Background и определяет фон на данном виде.
ClearColor
Свойство ClearColor позволяет задать режим удаления цветов на данном виде как значение из перечисления Teigha.GraphicsSystem.ClearColor.
ClientViewInfo
Свойство ClientViewInfo представляет экземпляр структуры Teigha.GraphicsSystem.ClientViewInfo, связанный с данным объектом View.
Device
Свойство Device представляет устройство, с которым связан данный вид.
EnableBackClip
Свойство EnableBackClip управляет включением задней плоскости отсечения.
EnableFrontClip
Свойство EnableFrontClip управляет включением передней плоскости отсечения.
EnableStereo
Свойство EnableStereo управляет режимом стереографического отображения на данном виде.
ExceededBounds
Свойство ExceededBounds позволяет определить, не превысили ли свои границы какие-либо параметры данного вида (положение камеры, цель камеры, поле зрения камеры, расстояние до передней/задней плоскости отсечения и дальность видимости) в процессе выполнения любого из следующих методов и свойств управления видом: SetView(), FrontClip, BackClip, Dolly(), Roll(), Orbit(), Zoom() и Pan() с момента предыдущего обращения к свойству ExceededBounds
FieldHeight
Свойство FieldHeight представляет высоту поля камеры данного вида.
FieldWidth
Свойство FieldWidth представляет ширину поля камеры данного вида.
FrontClip
Свойство FrontClip представляет удаление плоскости отсечения от точки цели камеры в направлении к точке положения камеры.
IsPerspective
Свойство IsPerspective позволяет определить, находится ли данный вид в режиме перспективного проецирования.
IsValid
Свойство IsValid позволяет определить, является ли действительной отображаемая на виде графика.
IsVisible
Свойство IsVisible позволяет определить, является ли данный вид видимым или скрытым.
LensLength
Свойство LensLength представляет фокусное расстояние камеры вида.
LinetypeScaleMultiplier
Свойство LinetypeScaleMultiplier представляет масштабный множитель типа линий.
LineweightToDcScale
Свойство LineweightToDcScale представляет коэффициент масштабирования при отображении толщины линий в данном объекте View.
Mode
Свойство Mode представляет текущий режим рендеринга, установленный для данного вида.
ObjectToDeviceMatrix
Свойство ObjectToDeviceMatrix представляет геометрическое преобразование из пространства модели в пространство устройства отображения.
Position
Свойство Position представляет текущую точку положения камеры.
ProjectionMatrix
Свойство ProjectionMatrix представляет проекционную матрицу преобразования из пространства вида в нормализованное пространство устройства.
ScreenMatrix
Свойство ScreenMatrix представляет матрицу преобразования координат из нормализованного пространства устройства в пространство экрана.
StereoParameters
Свойство StereoParameters представляет текущее сочетание значений магнитуды разделения видов и параллакса.
Target
Свойство Target представляет точку цели камеры данного вида.
UpVector
Свойство UpVector представляет текущее направление вертикали камеры данного вида.
UserGiContext
Свойство UserGiContext представляет контекст пользовательского графического интерфейса.
ViewingMatrix
Свойство ViewingMatrix представляет матрицу преобразования из мировой системы координат в пространство вида.
Viewport
Свойство Viewport представляет 2D-границы видового экрана в координатах устройства.
ViewportBorderProperties
Свойство ViewportBorderProperties предоставляет доступ к свойствам границ видового экрана.
ViewportBorderVisibility
Свойство ViewportBorderVisibility позволяет определить состояние видимости границ видового экрана.
ViewportClipRegion
Свойство ViewportClipRegion представляет текущую область обрезки данного вида.
VisualStyle
Свойство VisualStyle представляет текущий визуальный стиль данного вида.
VisualStyleId
Свойство VisualStyleId представляет идентификатор объекта визуального стиля данного вида.
WorldToDeviceMatrix
Свойство WorldToDeviceMatrix представляет преобразование координат из мирового пространства в пространство устройства (экрана).
