Свойства
Имя
Описание
ExtendedLightLength
Свойство ExtendedLightLength позволяет определить длину протяженного источника света.
ExtendedLightRadius
Свойство ExtendedLightRadius позволяет определить радиус протяженного источника света.
ExtendedLightShape
Свойство ExtendedLightShape позволяет определить форму протяженного источника света.
ExtendedLightWidth
Свойство ExtendedLightWidth определяет ширину протяженного источника света.
ShadowMapSize
Свойство ShadowMapSize позволяет определить размер теневой карты.
ShadowMapSoftness
Свойство ShadowMapSoftness позволяет определить размытость края тени.
ShadowSamples
Свойство ShadowSamples определяет количество лучей, используемых алгоритмом выборочных теней.
ShadowType
Свойство ShadowType определяет используемый метод расчета теней, отбрасываемых данным источником света.
ShadowsOn
Свойство ShadowsOn определяет, отбрасывает ли тени данный источник света.
ShapeVisibility
Свойство ShapeVisibility позволяет определить, видима ли форма источника света при рендеринге.
