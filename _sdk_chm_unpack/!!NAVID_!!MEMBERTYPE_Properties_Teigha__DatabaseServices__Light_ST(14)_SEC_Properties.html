Light Свойства Свойства
Light Свойства  Условные обозначения
Свойства
Имя
Описание
Attenuation
Свойство предоставляет доступ к информации о пределах затухания источника света.
AttenuationType
Свойство представляет тип затухания данного источника света.
Direction
Свойство представляет направление удаленного источника света.
EndLimitOffset
Свойство представляет конечный предел затухания источника света.
FalloffAngle
Свойство представляет полный угол светового конуса, геометрически определяющий спад освещенности для данного источника света.
GlyphDisplayType
Свойство представляет вариант отображения глифа для данного источника света.
HasTarget
Свойство позволяет проверить, отображает ли данный источник света ручку цели для выбора направления распространения света.
HotspotAngle
Свойство возвращает угол светового пучка источника света, определяющий самую яркую его часть (наиболее яркое пятно данного источника).
IlluminanceDistance
Свойство представляет дальность освещения для данного источника света.
Intensity
Свойство представляет интенсивность данного источника света.
IsOn
Свойство позволяет проверить, включен ли данный источник света.
IsPlottable
Свойство позволяет проверить, отображается ли этот источник света при печати.
LampColorPreset
Свойство представляет предустановленный цвет лампы данного источника света.
LampColorRGB
Свойство представляет значение RGB цвета лампы данного источника света.
LampColorTemp
Свойство представляет цветовую температуру источника в Кельвинах, если в свойстве Light.LampColorType задан тип LampColorType.Kelvin.
LampColorType
Свойство представляет тип цвета лампы, который определяет, как именно задается цвет лампы (цветовая температура в Кельвинах или предустановленный цвет лампы).
LightColor
Свойство представляет цвет данного источника света.
LightType
Свойство представляет тип данного источника света: DistantLight, PointLight, SpotLight или WebLight.
MapSize
Свойство представляет размер карты теней. Возможные значения: 64, 128, 256, 512, 1024, 2048 или 4096.
Name
Свойство представляет наименование данного источника света.
PhysicalIntensity
Свойство представляет физическую интенсивность источника света.
PhysicalIntensityMethod
Свойство представляет способ задания физической интенсивности данного источника света. Он определяет метод вычисления физической интенсивности света (по пиковой интенсивности, по потоку или по освещенности).
Position
Свойство представляет положение точечного источника света и прожектора.
Shadow
Свойство представляет параметры тени для данного источника света.
ShadowType
Свойство представляет тип тени для данного источника света.
Softness
Свойство представляет параметр мягкости тени для данного источника света.
StartLimitOffset
Свойство представляет начальный предел затухания для данного источника света.
TargetLocation
Свойство представляет положение цели данного источника света.
UseLimits
Свойство позволяет проверить, используются ли пределы в расчетах затухания света от источника.
WebFile
Свойство представляет полный путь к файлу световой сетки с гониометрическими данными источника света (файл IES).
WebRotation
Свойство представляет угловые смещения световой сетки, необходимые для ориентирования гониометрических данных относительно текущей ориентации источника света.
