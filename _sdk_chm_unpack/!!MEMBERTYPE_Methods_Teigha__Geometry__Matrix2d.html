AlignCoordinateSystem
Метод предоставляет матрицу, которая преобразует одну систему координат в другую. Каждая из систем координат указывается точкой начала и двумя осями.
Displacement
Метод формирует трансляционную часть матрицы преобразования на основе указанного вектора.
Equals
Метод предназначен для проверки на равенство данного объекта и указанного объекта.
GetDeterminant
Метод предоставляет детерминант данной матрицы.
GetHashCode
Метод предоставляет хеш-код данного объекта.
GetScale
Метод вычисляет масштабный коэффициент этой матрицы.
Inverse
Метод предоставляет матрицу, которая является обратной для данной матрицы.
IsConformal
Метод позволяет проверить, представляет ли данная матрица конформное линейное преобразование. Векторы в столбцах конформной матрицы равны по длине и взаимно ортогональны.

Multiply
Метод реализует левое умножение данной матрицы на указанную матрицу, являющуюся левым множителем.
PostMultiplyBy
Метод реализует правое умножение данной матрицы на указанную матрицу, являющуюся правым множителем.
PreMultiplyBy
Метод реализует левое умножение данной матрицы на указанную матрицу, являющуюся левым множителем.
Rotation
Метод предоставляет матрицу, которая реализует вращение вокруг указанной точки на заданный угол.
Scaling
Метод предоставляет матрицу равномерного масштабирования относительно указанной точки с заданным коэффициентом масштабирования.
ToArray
Метод позволяет получить данную матрицу в виде массива из девяти элементов. Запись элементов матрицы в массив выполняется построчно.
ToString

Transpose
