Описание

Делегат используется как сигнатура метода-обработчика для событий, связанных с операцией выбора вложенного объекта — например, при вызове метода GetNestedEntity() или внутри события PromptedForNestedEntity. Он позволяет получить доступ к результату выбора, включая идентификатор (ObjectId) выбранного элемента, точку ввода (PickedPoint) и матрицу преобразования блока. Эти данные предоставляются через параметр e, который представляет собой экземпляр класса PromptNestedEntityResultEventArgs. 

Этот тип делегата применяется при разработке надстроек, где требуется взаимодействовать с объектами, расположенными внутри блоков или внешних ссылок — например, для редактирования, анализа или построения на основе вложенных элементов.
