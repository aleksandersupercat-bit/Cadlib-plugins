Ниже перечислены свойства класса PlotStyleData.

Свойства
Имя
Описание
AdaptiveLinetype
Свойство AdaptiveLinetype позволяет определить, является ли тип линии адаптивным в данном объекте.
Color
Свойство Color представляет текущий цвет для данного объекта PlotStyleData.
ColorPolicy
Свойство ColorPolicy представляет битовые флаги, определяющие использование цвета стилем печати цвета для отображения объектов чертежа.
DitherOn
Свойство DitherOn позволяет определить, включено ли в настройках стиля печати использование дизеринга.
FillStyle
Свойство FillStyle представляет стиль заливки в данном объекте.
GrayScaleOn
Свойство GrayScaleOn позволяет определить, включен ли режим GrayScale.
LineEndStyle
Свойство LineEndStyle представляет текущий стиль концов линий, используемый в данном объекте.
LineJoinStyle
Свойство LineJoinStyle представляет текущий стиль соединения линий, используемый в данном объекте.
LinePatternSize
Свойство LinePatternSize представляет размер шаблона линии в данном объекте.
LineType
Свойство LineType представляет тип линии для данного объекта PlotStyleData.
Lineweight
Свойство Lineweight представляет вес (толщину) линии в данном объекте.
PhysicalPenNumber
Свойство PhysicalPenNumber представляет номер физического пера для данного объекта.
Screening
Свойство Screening представляет текущее значение скрининга в данном объекте.
VirtualPenNumber
Свойство VirtualPenNumber представляет номер виртуального пера для данного объекта.
