Описание

Метод позволяет создать трехмерное твердое тело вращением указанной области на заданный угол вокруг оси вращения. 

Ось вращения определяется проекцией линии, заданной точкой и вектором, на плоскость области region параллельно ее нормали. Таким образом, вращаемая область и ось вращения всегда компланарны. 

Область поворачивается на угол angleOfRevolution по модулю 2*Pi в радианах вокруг оси вращения. Положительное направление отсчета углов определяется правилом правой руки - по часовой стрелке, если смотреть вдоль оси вращения. 

Ось вращения должна быть допустимой для вращения области region:

  • Направление axisDir не может быть перпендикулярно плоскости области region,
  • Проекция оси вращения не может пересекать переданную в метод область region.

 

Действуют следующие ограничения:

  • Угол поворота, который по модулю меньше 1e-6, считается равным нулю.

 

Этот метод реализован только в среде пространственного моделирования.
