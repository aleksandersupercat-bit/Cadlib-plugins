PixelFormatInfo Свойства
Имя
Описание
AlphaOffset
Свойство AlphaOffset определяет битовое смещение для значения альфа-канала в пиксельном формате растрового изображения.
BitsPerPixel
Свойство BitsPerPixel определяет количество бит, которыми представлен один пиксель растрового изображения.
BlueOffset
Свойство BlueOffset определяет битовое смещение для значения синего канала в пиксельном формате растрового изображения.
GreenOffset
Свойство GreenOffset определяет битовое смещение для значения зеленого канала в пиксельном формате растрового изображения.
NumAlphaBits
Свойство NumAlphaBits представляет количество бит, которыми кодируется значение альфа-канала в формате растрового изображения.
NumBlueBits
Свойство NumBlueBits представляет количество бит, которыми кодируется значение синего канала в формате растрового изображения.
NumGreenBits
Свойство NumGreenBits представляет количество бит, которыми кодируется значение зеленого канала в формате растрового изображения.
NumRedBits
Свойство NumRedBits представляет количество бит, которыми кодируется значение красного канала в формате растрового изображения.
RedOffset
Свойство RedOffset представляет битовое смещение для значения красного канала в пиксельном формате растрового изображения.
