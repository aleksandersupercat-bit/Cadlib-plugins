Класс Xline представляет в nanoCAD тип объекта XLINE. Объект Xline — это линия, бесконечная в обоих направлениях.
Xrecord
Класс Xrecord — это класс для хранения и управления произвольными данными.
XrefBeginOperationEventArgs
Класс XrefBeginOperationEventArgs представляет данные для событий базы данных XrefBeginRestore, XrefBeginAttached и XrefBeginOtherAttached.
XrefComandeeredEventArgs
Класс XrefComandeeredEventArgs представляет данные для события XrefComandeered в базе данных.
XrefFullSubentityPath
Класс XrefFullSubentityPath уникальным образом определяет в базе данных подобъекты внешних ссылок.
XrefGraph
Класс XrefGraph представляет дерево внешних ссылок базы данных.
XrefGraphNode
Класс XrefGraphNode представляет в базе данных узлы графа внешних ссылок.
XrefPreXrefLockFileEventArgs
Класс XrefPreXrefLockFileEventArgs представляет данные для события XrefPreXrefLockFile в базе данных.
XrefRedirectedEventArgs
Класс XrefRedirectedEventArgs представляет данные для события XrefRedirected базы данных.
XrefSubCommandEventArgs
Класс XrefSubCommandEventArgs представляет данные для событий базы данных, связанных с подкомандами внешних ссылок (XrefSubCommandStart, XrefSubCommandEnd и XrefSubCommandAborted).
XrefVetoableSubCommandEventArgs
Класс XrefVetoableSubCommandEventArgs представляет данные для команды XrefVetoableSubCommand базы данных.
