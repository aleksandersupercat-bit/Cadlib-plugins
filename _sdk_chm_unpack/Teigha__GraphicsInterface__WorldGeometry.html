Описание

Класс WorldGeometry отвечает за помещение геометрии в графический кэш для последующего отображения. Объект этого класса является частью каждого объекта WorldDraw, передаваемого в метод WorldDraw() объекта, которому потребовалась генерация графики. Геометрия, созданная методами этого класса, используется во всех видовых экранах. Если требуется графика, специфичная для видового экрана, то вместо WorldDraw() необходимо использовать метод ViewportDraw(). 

Этот класс также предоставляет текущее геометрическое преобразование из системы координат объекта в пространство модели и позволяет пользователю определить пространственные границы объекта. 

В этом классе вся геометрия рассматривается в системе координат модели. Лишь только метод WorldGeometry.SetExtents() требует указатель на массив из двух точек в мировой системе координат. 

Класс WorldGeometry можно наследовать, чтобы создать свой класс с собственными реализациями различных геометрических функций. Такой объект может быть передан в метод WorldDraw() объекта пользовательского класса, производного от WorldDraw. Это дает вам возможность получать графическую информацию об объекте без прямого доступа к его базовому коду. Например, это оказывается полезным, когда требуется доступ к фасетной геометрии твердых тел.
