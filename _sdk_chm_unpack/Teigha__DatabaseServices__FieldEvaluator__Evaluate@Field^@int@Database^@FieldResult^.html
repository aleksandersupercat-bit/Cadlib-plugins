Описание

Метод предназначен для вычисления значения поля.

Замечания

Этот метод вызывается фреймворком поля всякий раз, когда значение поля требуется перевычислить. 

Поле вычисляется только тогда, когда заданная в этом поле опция вычисления соответствует контексту, в котором вызывается вычисление поля. Например, если опцией предусмотрено вычисление только вручную, то вычисление производится только при обновлении определенного поля или всех полей в чертеже вручную. 

Вычислитель должен перевычислить поле и задать полученное значение с помощью метода FieldResult.SetFieldValue() и статус ошибки с помощью метода FieldResult.setEvaluationStatus() в переданном объекте FieldResult, который, в свою очередь, задает их в объекте Field. 

Переданный указатель базы данных может быть нулевым, если поле не является резидентным в базе данных, и база данных не может быть получена в текущем контексте. Если же для вычисления поля база данных необходима, этот метод может задать статус ошибки в объекте FieldResult поля и вернуть ошибку. 

Если вычисление не удалось, кэшированный ранее результат в объекте Field можно оставить без изменений и использовать в качестве значения поля. В качестве альтернативы кэшированный результат можно удалить и заменить строкой ошибки (например, "#ERR"), которая будет отображаться как значение поля.
