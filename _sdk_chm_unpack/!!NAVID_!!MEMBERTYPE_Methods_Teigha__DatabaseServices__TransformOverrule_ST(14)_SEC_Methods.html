Описание
CloneMeForDragging
Метод проверяет необходимость создавать для операций перетаскивания клон указанного объекта.
Explode
Метод взрывает указанный в аргументе объект с получением множества более простых объектов.
GetTransformedCopy
Метод создает копию указанного объекта и применяет к этой копии заданное геометрическое 3D-преобразование.
HideMeForDragging
Метод проверяет, должен ли указанный объект быть скрытым для операций перетаскивания.
SetCustomFilter
Этот метод включает использование метода IsApplicable, чтобы устанавливать применимость объектного переопределения.
SetExtensionDictionaryEntryFilter
Метод включает предварительный поиск в словаре расширений для установления применимости объектного переопределения.
SetIdFilter
Метод включает предварительную фильтрацию объектов по их ObjectId прежде чем применять к ним объектное переопределение.
SetNoFilter
Метод отключает все методы предварительной фильтрации (SetCustomFilter, SetExtensionDictionaryEntryFilter, SetIdFilter, SetXDataFilter), а также метод IsApplicable, перед применением объектных переопределений.
SetXDataFilter
Метод включает использование расширенных данных в условии применимости объектного переопределения.
TransformBy
Метод предназначен для выполнения 3D-преобразования заданного объекта с использованием указанной матрицы преобразования.
