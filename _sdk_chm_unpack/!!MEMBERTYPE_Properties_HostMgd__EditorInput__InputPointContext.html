InputPointContext Свойства

Ниже перечислены свойства класса InputPointContext.

Свойства
Имя
Описание
CartesianSnappedPoint
Свойство предназначено для получения точки, привязанной по прямоугольной (декартовой) сетке, если в контексте ввода был активирован соответствующий режим привязки.
ComputedPoint
Свойство предназначено для получения или установки конечной вычисленной точки ввода, если она была определена в процессе обработки.
Document
Свойство предназначено для получения ссылки на документ, в котором произошло событие ввода точки.
DrawContext
Свойство предназначено для получения или установки контекста отрисовки, используемого при визуализации графики во время операции ввода точки.
GrippedPoint
Свойство предназначено для получения координат точки ручки (grip), если в контексте ввода активирован режим редактирования с помощью ручек и установлен соответствующий флаг.
History
Свойство предназначено для получения информации о типе привязки или операции, выполненной при вводе точки, на основе битовых флагов.
LastPoint
Свойство предназначено для получения последней введённой точки, если в истории ввода установлен соответствующий флаг.
ObjectSnapMask
Свойство предназначено для получения текущего режима объектной привязки (OSNAP), если точка была вычислена и активирована привязка к объекту.
ObjectSnapOverrides
Свойство предназначено для получения текущего режима объектной привязки, активированного пользователем вручную (например, через клавишу Shift + ПКМ), независимо от того, участвует ли он в вычислении точки.
ObjectSnappedPoint
Свойство предназначено для получения координат точки, к которой была выполнена объектная привязка (OSNAP), если таковая активирована и вычислена.
PointComputed
Свойство предназначено для определения, была ли успешно вычислена точка ввода на текущем этапе операции.
RawPoint
Свойство предназначено для получения исходной координаты курсора до применения привязок, фильтрации или выравнивания.
ToolTipText
Свойство предназначено для получения текста всплывающей подсказки, ассоциированной с текущим инструментом или операцией ввода точки.
