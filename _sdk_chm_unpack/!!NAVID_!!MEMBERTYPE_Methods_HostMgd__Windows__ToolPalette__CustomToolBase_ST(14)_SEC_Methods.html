Методы
Имя
Описание
BeginEdit
Метод начинает редактирование свойств инструмента. Вызывается фреймворком после вызова метода Load() перед отображением диалогового окна для редактирования свойств инструмента. Не реализован.
BeginMultipleEdit
Метод позволяет начать операцию группового редактирования сразу нескольких инструментов на панели инструментов. Не реализован.
CreateCommandTool
Метод вызывается фреймворком для стандартного инструмента при перетаскивании объекта на панель инструментов. Не реализован.
CreateFlyoutTool
Метод создает выпадающий инструмент, который может содержать пакеты других инструментов. Не реализован.
CreatePalette
Метод создает панель инструментов с заданным именем и добавляет в неё указанный каталог. Не реализован.
CreateShapeCatalog
Метод создает для единственной формы каталог, содержащий пакеты форм. Не реализован.
CreateStockTool
Метод создает именованный каталог для стандартных инструментов. Не реализован.
CreateTool
Это обзор перегрузок CreateTool.
Customize
Метод реализует настройку инструмента любого типа, поддерживаемого панелями инструментов. Не реализован.
DoModal
Метод отображает модальное диалоговое окно для взаимодействия с пользовательским инструментом. Не реализован.
DragEnter
Метод вызывается, когда перетаскиваемый объект стандартного инструмента попадает в область панели инструментов. Не реализован.
DragLeave
Метод вызывается, когда перетаскиваемый объект стандартного инструмента покидает область панели инструментов. Не реализован.
DragOver
Метод вызывается, когда перетаскиваемый объект стандартного инструмента движется в пределах области панели инструментов. Не реализован.
Drop
Метод вызывается, когда перетаскиваемый объект стандартного инструмента бросается на панель инструментов. Не реализован.
DropCallback
Метод определяет реакцию приложения, когда пользователь завершает операцию перетаскивания объекта чертежа из редактора nanoCAD на панель инструментов. Не реализован.
Dropped
Метод вызывается после того, как пользователь перетащил инструмент на панель инструментов с помощью пользовательской процедуры перетаскивания или IDrop. Не реализован.
Edit
Это обзор перегрузок Edit.
Editable
Метод позволяет управлять возможностью редактирования пользовательского инструмента на панели инструментов. Не реализован.
EndEdit
Метод завершает редактирование пользовательского инструмента с опцией отмены изменений. Не реализован.
EndMultipleEdit
Метод вызывается для завершения операции группового редактирования пользовательских инструментов на панели инструментов. Не реализован.
Execute
Метод запускает в действие логику пользовательского инструмента при его активации на панели инструментов. Не реализован.
ExecuteCallback
Метод реализует обратный вызов, используемый при выполнении логики пользовательского инструмента. Не реализован.
GetCommandString
Метод отображает однострочный поясняющий текст для команды, добавленной в контекстное меню методом GetContextMenu(). Не реализован.
GetContextMenu
Метод позволяет получить пользовательское контекстное меню, связанное с пользовательским инструментом на панели инструментов. Не реализован.
GetCustomPropertyCtrl
Метод предоставляет настраиваемый элемент управления ActiveX для отображения и редактирования свойств пользовательского инструмента. Не реализован.
GetData
Метод предоставляет COM-интерфейсы, необходимые для редактирования свойств пользовательского инструмента. Не реализован.
GetDisplayName
Метод предоставляет альтернативное или локализованное имя для свойства с заданным идентификатором. Не реализован.
GetDisplayString
Метод предоставляет отображаемый текст, связанный со свойством пользовательского инструмента. Не реализован.
GetDragDropContextInfo
Метод вызывается во время перетаскивания правой кнопкой мыши (drag-and-drop) пользовательского инструмента из панели инструментов. Не реализован.
GetDropTarget
Метод предназначен для получения интерфейса IDropTarget от инструмента. Не реализован.
GetMenuHelp
Метод вызывается для отображения всплывающей однострочной справки для команды, добавленной в контекстное меню методом GetContextMenu() инструмента. Не реализован.
GetPredefinedStrings
Метод предоставляет список предопределённых строковых значений для конкретного свойства пользовательского инструмента. Не реализован.
GetPredefinedValue
Метод вызывается для получения фактического значения свойства, когда пользователь выбирает один из предопределённых вариантов. Не реализован.
GetPropTextColor
Метод задает пользовательский цвет текста метки для свойства. Не реализован.
GetPropertyIcon
Метод позволяет добавлять пользовательские иконки для каждого свойства пользовательского инструмента. Не реализован.
GetPropertyWeight
Метод задает относительный вес (приоритет) категории или свойства на панели инструментов, влияющий на порядок отображения категорий и свойств в пользовательском интерфейсе. Не реализован.
GetStockTool
Метод позволяет получить ссылку на стандартный инструмент, связанный с данным пользовательским инструментом. Не реализован.
GetToolProperties
Метод предназначен для получения объекта, представляющего свойства пользовательского инструмента. Не реализован.
GetUnspecifiedString
Метод предоставляет текстовую строку со значением свойства, которое не указано явно или которое помечено как "не указано" (unspecified). Не реализован.
InvokeCommand
Метод предназначен для выполнения пользовательских команд, связанных с данным инструментом и выполняющих дополнительные действия с инструментом. Не реализован.
InvokeMenuCommand
Метод вызывается при активации пользовательского пункта контекстного меню, связанного с данным инструментом. Не реализован.
IsFullView
Метод вызывается, чтобы выяснить требуемый размер отображаемого свойства данного инструмента в расширенном (full) виде (например, как многострочный текст). Не реализован.
IsUnspecifiedAllowed
Метод позволяет определить, допустимо ли для указанного свойства использовать представление "не указано" (unspecified). Не реализован.
IsValueUnspecified
Метод позволяет определить, является ли текущим значением указанного свойства значение "не указано" (unspecified). Не реализован.
Load
Метод загружает данные инструмента из XML-узла в XmlElement перед использованием инструмента. Не реализован.
MapPropertyToPage
Метод выполняет сопоставление идентификатора COM-класса CLSID со страницей свойств в диалоговом окне для указанного свойства данного инструмента. Не реализован.
MultipleEdit
Метод предназначен для группового редактирования пользовательских инструментов. Не реализован.
New
Метод предназначен для создания нового пользовательского инструмента. Не реализован.
Refreshed
Метод вызывается после обновления данного пользовательского инструмента. Не реализован.
Save
Метод предназначен для сохранения данных элемента каталога в XML-узле. Не реализован.

Метод вызывается для инструмента, отображаемого в виде формы на выпадающей панели инструментов, когда выпадающий инструмент выполняется или перетаскивается. Не реализован.
SetToolProperties
Метод используется для инициализации или получения свойств инструмента. Не реализован.
SetUnspecified
Метод позволяет определить, использует ли указанное свойство представление "не указано" (unspecified). Не реализован.
ShowProperty
Метод позволяет динамически управлять отображением свойства инструмента. Не реализован.
ValidateEditChanges
Метод выполняет проверку изменений, внесённых пользователем в процессе редактирования свойств инструмента. Вызывается после нажатия кнопки "ОК" в редакторе свойств инструмента. Не реализован.
