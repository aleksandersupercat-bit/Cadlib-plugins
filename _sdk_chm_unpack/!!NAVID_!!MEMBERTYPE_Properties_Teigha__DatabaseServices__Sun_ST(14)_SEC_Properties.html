Sun Свойства Свойства
Sun Свойства  Условные обозначения
Свойства
Имя
Описание
Altitude
Свойство Altitude представляет высоту солнца над горизонтом.
Azimuth
Свойство Azimuth представляет азимут солнца.
DateTime
Свойство DateTime представляет дату и время, используемые для калькулятора солнечных углов.
Intensity
Свойство Intensity представляет значение интенсивности солнечного освещения.
IsDaylightSavingsOn
Свойство IsDaylightSavingsOn представляет статус флага учета перехода на летнее время, который используется калькулятором солнечных углов.
IsOn
Свойство IsOn представляет состояние флага учета солнечного освещения.
ShadowParameters
Свойство ShadowParameters представляет параметры теней, связанных с солнечным освещением.
SunColor
Свойство SunColor представляет цвет солнечного освещения.
SunDirection
Свойство SunDirection представляет направление от солнца к модели.
