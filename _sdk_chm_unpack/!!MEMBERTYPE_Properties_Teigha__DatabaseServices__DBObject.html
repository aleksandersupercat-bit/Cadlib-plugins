ClassID
Свойство предоставляет идентификатор класса данного объекта.
Database
Свойство предоставляет доступ к базе данных, в которой хранится данный объект.
Drawable
Свойство предоставляет доступ к графическому объекту, связанном с этим объектом базы данных.

Свойство предоставляет идентификатор словаря расширения этого объекта.
Handle
Свойство предоставляет идентификатор Handle данного объекта.
HasFields
Свойство предоставляет информацию, есть ли у объекта хотя бы одно поле.
HasSaveVersionOverride
Свойство предоставляет доступ к флагу, указывающему, что объект переопределяет способ сохранения объекта в dwg-файл.
Id
Свойство предоставляет идентификатор объекта в базе данных чертежа.
IsAProxy
Свойство предоставляет информацию, является ли данный объект прокси-объектом.

Свойство предоставляет информацию, происходит ли сейчас отмена всех изменений объекта.

Свойство предоставляет информацию, был ли переключен бит очистки объекта с момента его открытия.
IsErased
Свойство предоставляет информацию, был очищен ли объект.
IsModified
Свойство предоставляет информацию, был ли объект изменен.
IsModifiedGraphics
Свойство предоставляет информацию, был ли объект, отнаследованный от класса Entity, изменен.
IsModifiedXData
Свойство предоставляет информаицю, были ли изменены XData объекта.
IsNewObject
Свойство предоставляет информацию, был ли закрыт объект с момента его создания.
IsNotifyEnabled
Свойство предоставляет информацию, открыт ли объект в режиме OpenMode.ForNotify.
IsNotifying
Свойство предоставляет информацию, отправляет ли в данный момент объект уведомление о событии.
IsObjectIdsInFlux
Свойство предоставляет информацию, находится ли данный объект в процессе преобразования.
IsPersistent
Свойство предоставляет информацию о том, является ли объект резидентом базы данных чертежа.
IsReadEnabled
Свойство предоставляет информацию, открыт ли объект на чтение.
IsReallyClosing
Свойство предоставляет информацию о статусе закрытия объекта.
IsTransactionResident
Свойство предоставляет информацию об участии объекта в транзакции.
IsUndoing
Свойство предоставляет информацию об участии объекта в операции Undo.
IsWriteEnabled
Свойство предоставляет информацию о статусе объекта: открыт ли он на запись и посылает ли уведомления.
MergeStyle
Свойство позволяет установить нужный стиль слияния словарей.
ObjectBirthVersion
Свойство предоставляет набор параметров объекта в исходной версии.
ObjectId
Свойство предназначено для получения идентификатора этого объекта.
OwnerId
Свойство предназначено для получения идентификатора владельца объекта.
PaperOrientation
Свойство предназначено для получения значения свойства ориентации листа аннотационного объекта.
UndoFiler
Свойство предоставляет доступ к истории операций с объектом.
XData
Свойство предоставляет доступ к XData этого объекта.
