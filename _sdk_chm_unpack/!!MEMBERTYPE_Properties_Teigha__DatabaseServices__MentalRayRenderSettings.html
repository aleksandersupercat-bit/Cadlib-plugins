MentalRayRenderSettings Свойства

Ниже перечислены свойства класса MentalRayRenderSettings.

Свойства
Имя
Описание
DiagnosticBSPMode
Свойство DiagnosticBSPMode представляет используемый режим диагностики BSP.
DiagnosticGridMode
Свойство DiagnosticGridMode представляет параметры диагностической сетки.
DiagnosticMode
Свойство DiagnosticMode представляет текущий режим диагностической визуализации.
DiagnosticPhotonMode
Свойство DiagnosticPhotonMode представляет используемый режим фотонной диагностики.
EnergyMultiplier
Свойство EnergyMultiplier представляет энергетический множитель для управления величиной непрямого освещения.
ExportMIEnabled
Свойство ExportMIEnabled позволяет определить, следует ли экспортировать файл MI после рендеринга изображения.
ExportMIFileName
Свойство ExportMIFileName представляет используемое имя MI-файла.
FGRayCount
Свойство FGRayCount представляет количество лучей, испускаемых из каждой точки в режиме финальной сборки.
FGSampleRadius
Свойство FGSampleRadius представляет минимальный и максимальный радиусы выборки финальной сборки.
FGSampleRadiusState
Свойство FGSampleRadiusState позволяет определить, следует ли использовать заданные пользователем минимальный и максимальный радиусы финальной сборки.
FinalGatheringEnabled
Свойство FinalGatheringEnabled позволяет определить, включен ли режим финальной сборки.
FinalGatheringMode
Свойство FinalGatheringMode представляет текущий режим финальной сборки. Не реализовано.
GIPhotonsPerLight
Свойство GIPhotonsPerLight представляет среднее количество фотонов, испускаемых каждым источником света в режиме Global Illumination.
GISampleCount
Свойство GISampleCount представляет максимальное количество фотонов в области точки выборки рендеринга, которые используются при расчете глобального освещения.
GISampleRadius
Свойство GISampleRadius представляет радиус области выборки для испускания фотонов в режиме глобального освещения.
GISampleRadiusEnabled
Свойство GISampleRadiusEnabled позволяет определить, следует ли использовать заданный пользователем радиус фотона.
GlobalIlluminationEnabled
Свойство GlobalIlluminationEnabled позволяет определить, следует ли включать расчет глобального освещения.
LightLuminanceScale
Свойство LightLuminanceScale представляет физическую яркость света с интенсивностью 1.0 (в канделах). Не реализовано.
MemoryLimit
Свойство MemoryLimit представляет максимальный объем памяти, который может быть выделен для рендеринга.
PhotonTraceDepth
Свойство PhotonTraceDepth представляет максимальную глубину (уровень рекурсии) при трассировке фотонов.
RayTraceDepth
Свойство RayTraceDepth представляет максимальную глубину (уровень рекурсии) при трассировке лучей.
RayTracingEnabled
Свойство RayTracingEnabled позволяет определить, включена ли трассировка лучей.
Sampling
Свойство Sampling представляет минимальный и максимальный размеры выборки для оценки затенения пикселей при рендеринге.
SamplingContrastColor
Свойство SamplingContrastColor представляет пороговые нижние границы отсечения для цветовых каналов, ниже которых выборочные значения игнорируются.
SamplingFilter
Свойство SamplingFilter представляет параметры фильтрации для слияния нескольких выборочных значений в общий цвет пикселя.
ShadowMapsEnabled
Свойство ShadowMapsEnabled позволяет определить, следует ли активировать теневые карты.
ShadowMode
Свойство ShadowMode представляет используемый метод вычисления теней методом трассировки лучей.
ShadowSamplingMultiplier
Свойство ShadowSamplingMultiplier представляет текущее значение множителя выборки для теней. Не реализовано.
TileOrder
Свойство TileOrder представляет порядок тайлинга (укладки) фрагментов изображения при рендеринге.
TileSize
Свойство TileSize представляет размер фрагментов при тайлинге изображения в процессе рендеринга.
