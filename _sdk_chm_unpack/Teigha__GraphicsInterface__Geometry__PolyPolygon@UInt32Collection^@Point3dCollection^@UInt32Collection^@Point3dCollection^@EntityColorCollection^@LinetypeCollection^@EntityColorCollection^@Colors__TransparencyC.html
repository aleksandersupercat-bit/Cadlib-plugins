Возвращает

Возвращает false, если примитив успешно сохранен в графической базе данных. Возвращаемое значение true указывает, что операция была прервана, и приложение хочет вернуть управление как можно скорее.

Описание

Метод вводит несколько полигонов в контекст векторизации. Эти полигоны могут быть определены с настройкой различных свойств, таких как форма, непрозрачность, контуры и т.д. Метод может одновременно рисовать несколько одинаковых и несколько различных полигонов. Если какие-то параметры не указаны, они берутся из текущих свойств объекта SubEntityTraits.

Пример

Пример вызова метода (здесь квадратные скобки обозначают массив):

Копировать Код
polyPolygon ( [100, 7], [100 + 7 = 107 общее количество позиций], [4, 6], [4 + 6 = 10 всего точек], [outlinecolor0, outlinecolor1], [pattern0, pattern1], [fillcolor0, fillcolor1], [opacity0, opacity1] );

Этот вызов обеспечивает:

  • 100 одинаковых 4-точечных полигонов (индекс полигона 0) с параметрами outlinecolor0, pattern0, fillcolor0, opacity0,
  • 7 одинаковых 6-точечных полигонов (индекс полигона 1) с параметрами outlinecolor1, pattern1, fillcolor1, opacity1.
