Описание
Add
Метод реализует сложение данного вектора с заданным 3D-вектором.
AngleOnPlane
Метод определяет угол между ортогональной проекцией этого вектора на заданный плоский объект и нулевым базисным вектором v0 этого плоского объекта, который возвращает метод PlanarEntity.GetCoordinateSystem(p, v0, v1).
Convert2d
Метод позволяет получить представление данного трехмерного вектора в виде 2D-вектора в системе координат входного плоского объекта. Предполагается, что данный вектор параллелен заданной плоскости.
CrossProduct
Метод реализует векторное произведение данного вектора и входного 3D-вектора.
DivideBy
Метод реализует деление компонент данного вектора на входное вещественное значение.
DotProduct
Метод реализует скалярное произведение данного вектора с входным 3D-вектором.

Метод позволяет получить 3D-вектор, который является результатом зеркального отражения данного вектора относительно заданной плоскости.
MultiplyBy
Метод реализует умножение данного вектора на заданное число.
Negate
Метод создает вектор, обратный данному.
OrthoProjectTo
Метод строит 3D-вектор, который является ортогональной проекцией данного вектора на заданную плоскость, проходящую через начало координат.
ProjectTo
Метод строит трехмерный вектор, который является результатом параллельной проекции данного вектора на указанную плоскость в заданном направлении.
RotateBy
Метод создает трехмерный вектор, который образуется в результате вращения данного вектора вокруг заданной оси на заданный угол.
Subtract
Метод реализует вычитание указанного вектора из данного вектора.
ToArray
Метод позволяет создать массив типа double, содержащий значения координат X, Y и Z данного вектора.
ToString
Это обзор перегрузок ToString.
TransformBy
Метод реализует геометрическое преобразование данного вектора, которое определено входной матрицей Matrix3d.
