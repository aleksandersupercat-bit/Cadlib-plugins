Table Свойства  Условные обозначения
Свойства
Имя
Описание
BreakEnabled
Данное свойство хранит информацию о режиме разбиения.
BreakFlowDirection
Данное свойство предоставляет доступ к направлению разбиения таблицы.
BreakOptions
Данное свойство представляет опции разбиения таблицы.
Cells
Данное свойство предоставляет доступ к ячейкам объекта Table.
Columns
Данное свойство предоставляет доступ к коллекции столбцов объекта Table.
Direction
Данное свойство предоставляет доступ к единичному вектору горизонтального направления данной таблицы
FlowDirection
Данное свойство представляет направление следования таблицы от своей первой строки к последней.
HasSubSelection
Данное свойство позволяет определить, имеется ли в данной таблице подвыборка.
Height
Данное свойство представляет полную высоту данной таблицы.
HorizontalCellMargin
Данное свойство представляет размер горизонтальных полей ячейки объекта Table.
IsHeaderSuppressed
Устарело. Используйте функционал объекта Cell. Свойство позволяет подавлять или отменять подавление строки заголовков столбцов объекта Table.
IsTitleSuppressed
Устарело. Используйте функционал объекта Cell. Свойство позволяет подавлять или отменять подавление строки наименования объекта Table.
MinimumTableHeight
Данное свойство представляет доступ к минимальной общей высоте таблицы.
MinimumTableWidth
Данное свойство представляет доступ к минимальной общей ширине таблицы.
NumColumns
Данное свойство представляет количество столбцов в таблице. Устарело. Используйте Table.Columns.Count или Table.SetSize(int numRows, int numCols).
NumRows
Данное свойство представляет количество строк в таблице. Устарело. Используйте свойство Table.Rows.Count или Table.SetSize(int numRows, int numCols).
Rows
Данное свойство предоставляет доступ к коллекции строк объекта Table.
SubSelection
Данное свойство предоставляет доступ к диапазону ячеек в подвыборке.
TableStyle
Данное свойство предоставляет доступ к идентификатору ObjectId объекта TableStyle, на который ссылается объект Table.
VerticalCellMargin
Данное свойство представляет вертикальные поля ячеек объекта Table.
Width
Свойство предоставляет доступ к общей ширине данной таблицы.
