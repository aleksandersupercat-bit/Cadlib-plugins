Teigha.DatabaseServices.AssocArray  Условные обозначения
AssocArray Методы
Имя
Описание
AddSourceEntity
Метод добавляет указанный геометрический объект в список исходных элементов ассоциативного массива.
CreateArray
Метод позволяет создать объект AssocArray с заданными исходными объектами, базовой точкой и параметрами.
DeleteItem
Метод позволяет пометить указанный по индексу элемент ассоциативного массива для удаления или для отмены удаления.
Explode
Метод создает коллекцию ObjectId объектов, образованных в результате разбиения на подобъекты входного ассоциативного массива.
GetAssociativeArray
Метод позволяет найти ассоциативный массив AssocArray, в котором участвует заданный объект.
GetItemTransform
Метод извлекает матрицу преобразования, которая применяется к элементу ассоциативного массива с указанным локатором ItemLocator.
GetParameters
Метод позволяет получить объект параметров данного ассоциативного массива.
IsAssociativeArray
Метод позволяет проверить, является ли указанный объект ассоциативным массивом.
IsErased
Метод проверяет, помечен ли как удаленный указанный элемент ассоциативного массива.
RemoveSourceEntity
Метод удаляет исходный объект с заданным ObjectId и обновляет каждый элемент ассоциативного массива, которым управляет соответствующее действие.
ReplaceItems
Метод заменяет указанные по индексу элементы ассоциативного массива новыми объектами из входной коллекции с заданной базовой точкой.
ResetItems
Метод удаляет любые переопределения (положение, ориентация или замена объекта) для всех элементов данного ассоциативного массива.
TransformItemBy
Метод применяет заданное геометрическое 3D-преобразование к указанному элементу ассоциативного массива.
getItemLocators
Метод извлекает список локаторов элементов для указанного списка путей подобъектов данного ассоциативного массива.
getItems
Метод извлекает в массив локаторы элементов данного ассоциативного массива.
