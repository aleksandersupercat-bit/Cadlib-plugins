Свойство возвращает результат операции интерактивного ввода точки, выполненной пользователем.

Public ReadOnly Property Result As PromptPointResult

Возвращает

Возвращает экземпляр класса PromptPointResult, который содержит:

  • Status — статус завершения операции (OK, Cancel, Error и др.).
  • Value — значение типа Point3d, представляющее координаты точки в мировой системе координат (WCS), если операция завершена успешно.
  • StringResult — строковое представление введённой пользователем информации (если использовался клавиатурный ввод).

Если операция не была завершена или произошла ошибка, возвращается объект со статусом, отличным от OK.

Описание

Свойство предоставляет доступ к объекту типа PromptPointResult, который содержит информацию о статусе и результате ввода точки (Point3d) через такие методы, как GetPoint(), GetCorner() и другие. Оно используется, например, внутри событий PromptedForCorner, чтобы получить координаты точки после завершения пользовательского ввода. Свойство доступно только для чтения и обновляется автоматически при окончании операции.
