Свойство Double представляет значение типа double данного объекта.

Свойство Float представляет значение типа float данного объекта.

Свойство Uchar представляет значение типа unsigned char данного объекта.
Uint
Свойство Uint представляет значение типа unsigned int данного объекта.

Свойство Ulong представляет значение типа unsigned long данного объекта.
