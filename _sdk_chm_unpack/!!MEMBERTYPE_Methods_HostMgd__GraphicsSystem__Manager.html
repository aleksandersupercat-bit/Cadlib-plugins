Методы
Имя
Описание
CreateAutoCADDevice
Метод создает объект Teigha.GraphicsSystem.Device с заданным дескриптором окна, в котором отображается чертеж.
CreateAutoCADModel
Метод создаёт объект Teigha.GraphicsSystem.Model, представляющий коллекции отображаемых объектов платформы ODA Platform.
CreateAutoCADOffScreenDevice
Метод создаёт новое устройство как объект Teigha.GraphicsSystem.Device для отрисовки объектов чертежа в невидимом режиме без визуализации в каком-либо отображаемом окне графического интерфейса.
CreateAutoCADView
Метод создаёт объект Teigha.GraphicsSystem.View и добавляет в этот вид переданный отображаемый объект Drawable.
CreateAutoCADViewport
Метод создаёт объект Teigha.GraphicsSystem.View и добавляет в него объект Drawable, соответствующий указанной записи в таблице видовых экранов ViewportTableRecord.
GetDBModel
Метод предоставляет модель графической системы, связанную с базой данных чертежа.
GetGUIDevice
Метод позволяет получить устройство, связанное с окном графического интерфейса nanoCAD.
GetGsModel
Метод предоставляет модель GS, которая соответствует заданным режиму и подрежиму отрисовки временной графики, а также идентификационному номеру видового экрана.
GetGsView
Метод позволяет получить объект View графической системы для указанного видового экрана.
SetViewFromViewport
Метод позволяет переопределить заданный вид GS, используя параметры указанного видового экрана.

Это обзор перегрузок SetViewportFromView.
