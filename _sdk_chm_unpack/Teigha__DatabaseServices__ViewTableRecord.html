Описание

Класс предназначен для хранения определения именованного вида. Определение именованного вида представляет из себя набор настроек вида. Именованный вид всегда относится к какому-либо листу. 

Управление настройками именованного вида возможно как через API, так и через пользовательский интерфейс с помощью команды ВИД -> Свойства вида. Все настройки именованных видов, которые есть в пользовательском интерфейсе, также доступны и через API в виде свойств класса ViewTableRecord. 

Для создания нового именованного вида необходимо обязательно указать его название. Все остальные свойства имеют значения по умолчанию.

Пример

В качестве примера приведена команда ADDNAMEDVIEW, которая создает именованный вид в чертеже для пространства модели и делает его активным.

public partial class Commands { [Teigha.Runtime.CommandMethod("AddNamedView")] public void AddNamedView() { // Получение ссылки на активный документ HostMgd.ApplicationServices.Document doc = HostMgd.ApplicationServices.Application.DocumentManager.MdiActiveDocument; // Получение ссылки на редактор докумена HostMgd.EditorInput.Editor ed = doc.Editor; // Получение ссылки на базу данных документа Teigha.DatabaseServices.Database db = doc.Database; // Создаем новый именованный вид, который по умолчанию будет относиться к пространству модели, // если не указать иное Teigha.DatabaseServices.ViewTableRecord newView = new Teigha.DatabaseServices.ViewTableRecord(); newView.Name = "newView"; // Начало транзакции с базой данных документа using (Teigha.DatabaseServices.Transaction trans = db.TransactionManager.StartTransaction()) { // Открываем таблицу именованных видов для записи Teigha.DatabaseServices.ViewTable viewTable = trans.GetObject(db.ViewTableId, Teigha.DatabaseServices.OpenMode.ForWrite) as Teigha.DatabaseServices.ViewTable; // Добавляем новый вид в таблицу именованных видов. // Той же операцией он добавится и в базу данных чертежа try { viewTable.Add(newView); } catch (Teigha.Runtime.Exception ex) { if (ex.ErrorStatus == Teigha.Runtime.ErrorStatus.DuplicateRecordName) { ed.WriteMessage($"Вид с названием {newView.Name} уже существует.");} } // Устанавливаем новому виду изометрический виду СВ изометрию newView.ViewDirection = new Teigha.Geometry.Vector3d(1, 1, 1); // Устанавливаем высоту и ширину вида, это влияет на масштаб // отображения чертежа на экране newView.Width = 90000; newView.Height = 37000; // Сохраняем именованный вид в чертеже, завершив транзакцию trans.Commit(); } // Устанавливаем новый именованный вид текущим string[] strings = { "View", "_restore", newView.Name }; ed.Command(strings); } }
