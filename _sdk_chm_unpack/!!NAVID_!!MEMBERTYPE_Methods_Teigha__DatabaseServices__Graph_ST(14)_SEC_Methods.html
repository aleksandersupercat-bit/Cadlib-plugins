Методы
Методы
Имя
Описание
AddEdge
Метод предназначен для добавления к данному объекту Graph нового ориентированного ребра с указанными начальным и конечным узлами.
AddNode
Метод предназначен для добавления в данный граф нового узла GraphNode.
BreakCycleEdge
Метод предназначен для разрыва цикла на конкретном ребре графа с последующим обновлением информации о циклах в данном графе.
ClearAll
Метод предназначен для очистки битовых флагов во всех узлах графа.
Create
Метод предназначен для создания объекта Graph на базе указанного объекта.
DelNode
Метод предназначен для полного удаления конкретного узла из данного графа.
FindCycles
Метод предназначен для выявления всех узлов данного графа, участвующих в циклах.
GetOutgoing
Метод позволяет получить коллекцию всех исходящих узлов данного графа.
Node
Метод предназначен для получения узла данного графа по его индексу.
Reset
Метод предназначен для удаления из данного графа всех узлов, включая узлы циклов, что делает граф пустым.
SetNodeGrowthRate
Метод позволяет задать количество узлов, под которые графу одномоментно выделяется память.
