Методы
Методы
Имя
Описание
BooleanOperation
Метод предназначен для выполнения логических операций между данным объектом и другим твердым телом (объединение, пересечение, вычитание).
ChamferEdges
Метод предназначен для создания фасок на ребрах данного твердого тела.
CheckInterference
Метод позволяет проверить, пересекается ли данный объект с другим твердым телом.
CleanBody
Метод позволяет удалить все избыточные ребра и грани, которые не являются необходимыми для обеспечения топологии данного твердого тела.
ConvertToBrepAtSubentPaths
Метод позволяет преобразовать отдельные элементы истории построения параметризованного типа в простое граничное представление без параметрической информации или данных о типе. В текущей версии метод не реализован.
CopyEdge
Метод позволяет сделать копию указанного ребра данного объекта в виде отдельного объекта (экземпляра класса Line, Circle, Arc, Ellipse, Spline).
CopyFace
Метод позволяет сделать копию указанной грани в виде отдельного объекта класса Region или Body.
CreateBox
Метод позволяет создать 3D-тело в виде прямоугольного параллелепипеда заданного размера, ребра которого выровнены по осям мировой системы координат (WCS), с центром в начале системы координат WCS.
CreateExtrudedSolid
Это обзор перегрузок CreateExtrudedSolid.
CreateFrom
Метод предназначен для создания трехмерного твердого тела из указанного геометрического объекта.
CreateFrustum
Метод создаёт усеченный конус, ориентированный по осям мировой системы координат (WCS), с центром основания в начале координат. В зависимости от входных параметров метода геометрия получаемого твердого тела может иметь вид круглого или эллиптического цилиндра, круглого или эллиптического конуса, круглого или эллиптического усеченного конуса.
CreateLoftedSolid
Метод предназначен для создания трехмерного твердого тела методом лофта - вытягиванием по сечениям.
CreatePyramid
Метод предназначен для создания твердого тела в виде пирамиды с указанными параметрами.
CreateRevolvedSolid
Это обзор перегрузок CreateRevolvedSolid.
CreateSphere
Метод создает сферу заданного радиуса с центром в начале мировой системы координат.
CreateSweptSolid
Это обзор перегрузок CreateSweptSolid.
CreateTorus
Метод предназначен для создания твердого тела в виде тора в мировой системе координат (WCS) c осью z в качестве оси вращения и с центром в начале координат.
CreateWedge
Метод создает трехмерный твердотельный клин, выровненный по осям мировой системы координат (WCS).
Extrude
Метод создает трехмерное твердое тело выдавливанием указанной области на заданную высоту и с заданным углом наклона.
ExtrudeAlongPath
Метод создает трехмерное твердое тело путем выдавливания указанной области вдоль заданной траектории.
ExtrudeFaces
Метод позволяет выдавить массив граней твердого тела на указанную длину с заданным углом наклона.
ExtrudeFacesAlongPath
Метод позволяет выдавить группу указанных граней твердого тела вдоль заданной траектории.
FilletEdges
Метод предназначен для создания скруглений на ребрах данного твердого тела.
GetSection
Метод предназначен для построения области Region, являющейся результатом сечения твердого тела указанной плоскостью.
GetSubentityColor
Метод позволяет получить цвет указанного подобъекта данного твердого тела.
GetSubentityMaterial
Метод позволяет получить материал для указанного подобъекта.
GetSubentityMaterialMapper
Метод позволяет для указанного подобъекта получить объект Mapper, который обеспечивает сопоставление материалов.
ImprintEntity
Метод реализует операцию клеймения данного твердого тела указанным объектом.
OffsetBody
Метод реализует смещение всех граней данного твердого тела на указанное расстояние.
OffsetFaces
Метод позволяет смещать указанные грани твердого тела на заданное расстояние.
ProjectOnToSolid
Метод предназначен для создания объектов проецированием указанного объекта в указанном направлении на данное 3D-тело. Созданные объекты не являются резидентными в базе данных.
RemoveFaces
Метод предназначен для удаления указанных граней из данного твердого тела.
Revolve
Метод позволяет создать трехмерное твердое тело вращением указанной области на заданный угол вокруг заданной оси.
SeparateBody
Метод позволяет разделить данное 3D-тело на массив твердых тел, представляющих дополнительные непересекающиеся объемы.
SetSubentityColor
Метод позволяет задать цвет указанной подобъекту грани или ребра.
SetSubentityMaterial
Метод позволяет назначить указанному подобъекту данного 3D-тела новый материал.
SetSubentityMaterialMapper
Метод позволяет задать указанному подобъекту новый объект Mapper для сопоставления материалов.
ShellBody
Метод позволяет преобразовать твердое тело в тонкостенную твердотельную оболочку путем смещения граней исходного тела.
Slice
Это обзор перегрузок Slice.
StlOut
Метод предназначен для записи представления данного твердого тела в формате стереолитографии (STL) в файл с указанным именем.
TaperFaces
Метод реализует сужение указанных граней твердого тела на заданный угол относительно базовой плоскости, определяемой заданными базовой точкой и вектором.
TransformFaces
Метод предназначен для геометрического преобразования указанных граней данного твердого тела, заданного матрицей преобразования.
