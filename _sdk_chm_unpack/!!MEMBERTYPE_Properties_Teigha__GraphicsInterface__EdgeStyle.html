EdgeStyle Свойства
Послать отзыв

Ниже перечислены свойства класса EdgeStyle.

Свойства
Имя
Описание
CreaseAngle
Свойство CreaseAngle определяет пороговый угол между смежными гранями, при превышении которого ребро отображается как "острое" (на изломе).
EdgeColor
Свойство EdgeColor определяет цвет отображаемых ребер.
EdgeModel
Свойство EdgeModel определяет модель для отображения ребер.
EdgeModifiers
Свойство EdgeModifiers определяет текущие модификаторы режима отображения рёбер.
EdgeStyleApply
Свойство EdgeStyleApply определяет тип геометрии, к которой следует применять стиль рёбер.
EdgeVisibility
Свойство EdgeVisibility определяет текущий режим видимости рёбер модели.
EdgeWidth
Свойство EdgeWidth определяет толщину отображаемых рёбер.
HaloGap
Свойство HaloGap представляет текущее значение коэффициента недовода рёбер (HaloGap) к рёбрам объекта, скрытого другим объектом.
HidePrecision
Свойство HidePrecision позволяет определить, допустимо ли в данном стиле снижать точность отображения ребер.
IntersectionColor
Свойство IntersectionColor позволяет определить цвет рёбер на пересечении графических объектов.
IntersectionLinetype
Свойство IntersectionLinetype определяет тип линий для ребер в области пересечения графических объектов.
Isolines
Свойство Isolines определяет количество изолиний, отображаемых на поверхностях объектов.
Jitter
Свойство Jitter определяет текущую настройку джиттера для получения эффекта легкой неровности при отрисовке рёбер.
ObscuredColor
Свойство ObscuredColor определяет цвет скрытых рёбер.
ObscuredLinetype
Свойство ObscuredLinetype определяет тип линий, используемый при отрисовке скрытых рёбер.
Opacity
Свойство Opacity определяет непрозрачность рёбер.
Overhang
Свойство Overhang определяет поддерживаемое данным стилем удлинение рёбер за пределы начальной и конечной точек.
SilhouetteColor
Свойство SilhouetteColor определяет цвет для отрисовки силуэтных рёбер объектов.
SilhouetteWidth
Свойство SilhouetteWidth определяет толщину силуэтных рёбер объектов.
