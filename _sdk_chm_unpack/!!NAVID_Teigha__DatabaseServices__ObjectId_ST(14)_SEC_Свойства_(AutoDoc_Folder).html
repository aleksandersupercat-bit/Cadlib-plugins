Teigha.DatabaseServices.ObjectId  Условные обозначения
ObjectId Свойства
Имя
Описание
Database
Свойство предоставляет доступ к базе данных, в которой ID объекта является резидентным. Если идентификатор объекта равен NULL, то свойство возвращает NULL.
Handle
Свойство предоставляет доступ к дескриптору объекта с данным ObjectId.
IsEffectivelyErased
Свойство определяет, удален или нет объект, связанный с этим ObjectId, а также не удален ли кто-либо из владельцев в иерархии владения этим объектом.
IsErased
Свойство возвращает true, если объект, связанный с данным ObjectId, удален.
IsNull
Свойство возвращает true, если ObjectId данного объекта является нулевым.
IsResident
Свойство возвращает значение true, если объект, связанный с этим ObjectId, находится в памяти.
IsValid
Свойство определяет, связан ли ObjectId с базой данных, которая в данный момент загружена в память. Если ObjectId связан с базой данных, которая больше недоступна (другими словами, была удалена), то возвращается значение false.
NonForwardedHandle
Если данный ObjectId находится в базе данных внешних ссылок, а связанный с ним объект, был перенаправлен в главную базу данных, то значением этого свойства является дескриптор (handle) в базе данных внешних ссылок. Если объект, связанный с этим ObjectId, не из базы данных внешних ссылок или не перенаправлен, то это свойство аналогично свойству ObjectId.Handle.
Null
Свойство обеспечивает доступ к нулевому ObjectId. Данное свойство позволяет использовать более компактные выражения и помогает компилятору генерировать более эффективный код (если он знает, что ObjectId нулевой, то нет необходимости искать объект Null).
ObjectClass
Свойство предоставляет доступ к RXClass объекта, который ассоциирован с ObjectId, таким образом гарантируя, что результат может быть безопасно разыменован и корректно сравнен с любым классом, производным от DBObject.
ObjectLeftOnDisk
OldId
Свойство устарело, вместо него следует использовать свойство OldIdPtr.
OldIdPtr
Свойство предоставляет доступ к значению ObjectId в старом формате - как к длинному целому (он представляет собой массив из двух long, первый элемент которого содержит то же значение, что и ObjectId).
OriginalDatabase
Если этот ObjectId находится в базе данных внешних ссылок (xref database), и связанный с ним объект был перенаправлен в главную базу данных, то данное свойство возвращает указатель на базу данных внешних ссылок. Если связанный с этим ObjectId объект не из базы данных внешних ссылок или не был перенаправлен, то значение этого свойства совпадает с ObjectId.Database.
