Возвращает

Возвращает код варианта вертикального выравнивания текста допуска. Тип int.

Описание

Свойство позволяет выбрать вариант вертикального выравнивания текста допуска относительно текста номинального размера. 

Свойство принимает следующие значения:

Значение
Описание
0
Снизу.
1
Посередине.
2
Сверху.

Это свойство имеет эффект только в том случае, когда отображение допуска включено (т.е. свойство Dimtol имеет значение true).
