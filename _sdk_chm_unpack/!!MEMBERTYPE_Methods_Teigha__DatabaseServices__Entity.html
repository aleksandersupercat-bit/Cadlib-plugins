GetGraphicsMarkersAtSubentityPathIntPtr
Метод предназначен для возврата всех маркеров графической системы (GS-маркеров) по путям подобъектов для сложных сущностей, которые поддерживают связывание целочисленного идентификатора с каждым графическим примитивом или группами примитивов во время отрисовки. Объекты чертежа используют механизм маркеров графической системы для отрисовки самих себя, при этом каждый графический примитив или группа примитивов связываются с уникальным целочисленным идентификатором, называемым GS-маркером. С помощью специальных механизмов выбора GS-маркер выбранного на экране подобъекта может быть получен и использован для различных целей, таких как выделение/снятие выделения, скрытие/отображение подобъекта и других операций.

GetPlane

GetStretchPoints
Метод предназначен для добавления в указанный массив stretchPoints координат точек растягивания геометрического объекта, используемых при выполнении команды растяжения в nanoCAD. Метод учитывает, что массив может содержать точки других объектов, поэтому новые точки должны добавляться, а не заменять существующие.
GetSubentity

GetSubentityGeometricExtents
Метод предназначен для получения трёхмерных пределов (границ) указанного геометрического подобъекта в виде параллелепипеда, ориентированного по осям WCS.

GetTransformedCopy
Метод создает копию геометрического объекта и применяет к ней заданное преобразование с помощью матрицы transform, после чего возвращает преобразованный клон.
Внимание: для сложных объектов, таких как полилинии, метод может привести к некорректному состоянию чертежа из-за разделяемого владения вложенными объектами.

IsContentSnappable
Метод проверяет, должен ли блок-контейнер участвовать в поиске точек объектной привязки (osnaps).
List
Метод List() вызывается из nanoCAD при выборе геометрического объекта в команде LIST и выводит информацию о нём в командную строку. Он может быть переопределён в производных классах для добавления специфичных сведений, предварительно вызвав реализацию родительского класса.
MoveGripPointsAt
Это обзор перегрузок MoveGripPointsAt.
MoveGripPointsAtSubentityPaths
Метод применяет смещение, заданное вектором offset, к выбранным точкам растяжения геометрического объекта, определённым в массиве appData, и позволяет точно указать, какие подобъекты участвуют в операции с помощью массива subPaths. Порядок и значения индексов определяются согласованной реализацией метода GetStretchPoints() в производных классах.
MoveStretchPointsAt
Метод применяет заданное смещение к выбранным точкам растяжения геометрического объекта, определяемым индексами из массива indices. Он вызывается при выполнении команды растяжения в nanoCAD или может быть использован в приложениях для программного изменения формы объекта.
RecordGraphicsModified
Метод устанавливает, должно ли графическое представление геометрического объекта обновляться при его закрытии.
SaveAs
Метод используется для генерации графических примитивов при сохранении геометрического объекта в виде прокси-объекта с помощью nanoCAD, обеспечивая его корректное отображение при последующей загрузке. Он вызывается автоматически при сохранении и позволяет переопределять логику сохранения графики через производные классы WorldDraw, WorldGeometry и SubEntityTraits.
SetDatabaseDefaults
Это обзор перегрузок SetDatabaseDefaults.
SetDragStatus
Метод вызывается при изменении состояния операции перетаскивания геометрического объекта, например, при её начале, завершении или отмене, и использует параметр status для получения информации о текущем состоянии операции.
SetGripStatus
Метод вызывается перед удалением объекта ручки (GripStatus) и связанных с ним точек, чтобы геометрический объект мог выполнить необходимые действия по управлению ресурсами.
SetLayerId
Метод устанавливает идентификатор слоя, к которому должен принадлежать объект по имени слоя. Также мтеод поддерживает установку на скрытый слой, если параметр allowHidden имеет значение true. Данный метод применим только к определённым встроенным типам объектов, таким как 2dPolyline, 3dPolyline, PolyFaceMesh и PolygonMesh.
SetPropertiesFrom
Метод копирует значения таких свойств, как цвет, слой, тип линии, масштаб линии и видимость, из указанного геометрического объекта entityPointer в текущий объект. Этот метод позволяет быстро передать графические параметры от одного объекта к другому.
SetSubentityGripStatus
Метод вызывается перед удалением ручек подобъектов, чтобы геометрический объект мог выполнить необходимые действия по управлению ресурсами, или после их удаления для уведомления об окончании сессии редактирования.
TransformBy
Метод применяет указанную матрицу преобразования к геометрическому объекту, изменяя его положение, ориентацию или размер в соответствии с заданным преобразованием. Реализация метода в производных классах позволяет определять поддерживаемые типы преобразований и обеспечивает корректное отображение объекта после их применения.
TransformSubentityPathsBy
Метод применяет указанную матрицу преобразования к одному или нескольким геометрическим подобъектам текущего объекта, изменяя их положение, ориентацию или размер. Реализация метода позволяет разработчику определять поддерживаемые типы преобразований и способ их применения к подобъектам.
