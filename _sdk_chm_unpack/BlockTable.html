Статья о том, как программно посмотреть перечень всех блоков чертежа.

Описание

В чертеже есть два типа блоков - служебные и пользовательские. Служебные блоки представляют собой блоки, обеспечивающие корректную работу поатформы и хранение данных чертежа. Пользовательские блоки - это блоки, которые создает пользователь в чертеже. В статье приведен пример о том, как посмотреть перечень всех блоков чертежа: служебных и пользовательских. У пользовательских блоков название начинается с символа *.

Пример

 

В качестве примера работы с таблицей блоков приведена команда BlockTableReading, которая открывает таблицу блоков и перебирает её содержимое, выводя в команданую строку название каждого блока и его тип.

C#
Копировать Код
public partial class Commands { [Teigha.Runtime.CommandMethod("BlockTableReading")] public void BlockTableUse() { // Получение ссылки на активный документ HostMgd.ApplicationServices.Document doc = HostMgd.ApplicationServices.Application.DocumentManager.MdiActiveDocument; // Получение ссылки на редактор докумена HostMgd.EditorInput.Editor ed = doc.Editor; // Получение ссылки на базу данных документа Teigha.DatabaseServices.Database db = doc.Database; // Начало транзакции с базой данных документа using (Teigha.DatabaseServices.Transaction trans = db.TransactionManager.StartTransaction()) { // Открываем таблицу блоков для чтения Teigha.DatabaseServices.BlockTable blockTable = trans.GetObject(db.BlockTableId, Teigha.DatabaseServices.OpenMode.ForRead) as Teigha.DatabaseServices.BlockTable; // Перебираем таблицу блоков в цикле foreach(Teigha.DatabaseServices.ObjectId btr in blockTable) { // Открываем каждую запись таблицы Teigha.DatabaseServices.BlockTableRecord obj = trans.GetObject(btr, Teigha.DatabaseServices.OpenMode.ForRead) as Teigha.DatabaseServices.BlockTableRecord; // Выводим в командную строку ed.WriteMessage($"nTable record name and type: {obj.Name} , {obj.GetType()} "); } trans.Abort(); } } }
