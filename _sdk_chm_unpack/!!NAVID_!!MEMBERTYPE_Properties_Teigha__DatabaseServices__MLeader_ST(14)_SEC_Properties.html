MLeader Свойства Свойства
MLeader Свойства  Условные обозначения
Свойства
Имя
Описание
ArrowSize
Свойство представляет размер стрелки для данного объекта MLeader.
ArrowSymbolId
Свойство представляет идентификатор объекта символа стрелки для данного объекта MLeader.
BlockColor
Свойство представляет цвет, используемый для отрисовки содержимого блока.
BlockConnectionType
Свойство представляет вариант соединения блока с линиями выноски в данном объекте MLeader.
BlockContentId
Свойство представляет идентификатор ObjectId блока, на который ссылается данный объект MLeader.
BlockPosition
Свойство представляет точку вставки блока, на который ссылается данный объект MLeader.
BlockRotation
Свойство представляет угол поворота блока, используемого данным объектом MLeader.
ContentType
Свойство представляет тип содержимого данного объекта MLeader.
DoglegLength
Свойство представляет длину полки в данном объекте MLeader.
EnableAnnotationScale
Свойство позволяет проверить, включено ли масштабирование аннотаций для данной мультивыноски.
EnableDogleg
Свойство позволяет контролировать отображение полки на выносной линии мультивыноски.
EnableFrameText
Свойство позволяет проверить видимость рамки вокруг текста у данной мультивыноски.
EnableLanding
Свойство должно управлять отображением зазора между концом линии полки и содержимым мультивыноски. В текущей реализации платформы всегда считается равным true.
ExtendLeaderToText
Свойство позволяет проверить, продлеваются ли горизонтальные линии выносок автоматически до текста.
LandingGap
Свойство представляет величину зазора между концом линии полки и связанным многострочным текстом данной мультивыноски.
LeaderCount
Свойство представляет количество выносных линий у этой мультивыноски. В текущей версии не реализовано. Рекомендуется использовать метод GetLeaderIndexes.
LeaderLineColor
Свойство представляет цвет линии выноски у данного объекта MLeader.
LeaderLineCount
Свойство представляет количество выносных линий у данного объекта MLeader. В текущей версии не реализовано. Используйте метод GetLeaderLineIndexes.
LeaderLineType
Свойство представляет тип линий выноски (прямая, сплайн, невидимая).
LeaderLineTypeId
Свойство представляет идентификатор ObjectId типа линии, используемого для отрисовки выносных линий данного объекта MLeader.
LeaderLineWeight
Свойство представляет толщину (вес) линий, которая используется данным объектом MLeader.
MLeaderStyle
Свойство представляет идентификатор ObjectId используемого стиля мультивыноски.
MText
Свойство представляет многострочный текст, связанный с данной мультивыноской.
Normal
Свойство представляет вектор нормали плоскости, в которой располагается данная мультивыноска.
TextAlignmentType
Свойство представляет тип выравнивания многострочного текста данного объекта MLeader.
TextAngleType
Свойство представляет выбор угла для размещения многострочного текста относительно последнего сегмента линии выноски.
TextAttachmentDirection
Свойство представляет направление полки линий выноски данного объекта MLeader.
TextAttachmentType
Свойство представляет вариант расположения точки прикрепления выносной линии к многострочному тексту.
TextColor
Свойство представляет цвет многострочного текста данного объекта MLeader.
TextHeight
Свойство представляет текущую высоту многострочного текста.
TextLocation
Свойство представляет точку привязки многострочного текста данного объекта MLeader.
TextStyleId
Свойство представляет идентификатор ObjectId текстового стиля, который применяется к многострочному тексту данной мультивыноски.
ToleranceLocation
Свойство представляет точку привязки содержимого допуска данной мультивыноски.
blockScale
Свойство представляет масштаб отрисовки блока, используемого данным объектом MLeader.
