Идентификатор объекта, к которому требуется получить доступ. Тип ObjectId.

Данный флаг определяет, следует ли получать доступ к объекту, если он был удалён. Тип bool.

Возвращает дескриптор полученного объекта. Тип DBObject.

Описание

Метод обеспечивает доступ к объекту с указанным идентификатором в указанном режиме доступа. Аргумент openErased делает возможным доступ к удаленным объектам, если передать с ним значение true. Если передать значение false, попытка доступа к удалённому объекту приведёт к получению кода ошибки. 

Используется обращение к методу Open() транзакции верхнего уровня с передачей ему всех полученных аргументов. Доступ к объектам возможен только в рамках транзакции верхнего уровня, поэтому этот метод избавляет от необходимости сохранять указатель на транзакцию верхнего уровня или использовать для этого свойство TransactionManager.TopTransaction. Рекомендуется проверять валидность полученного объекта.
