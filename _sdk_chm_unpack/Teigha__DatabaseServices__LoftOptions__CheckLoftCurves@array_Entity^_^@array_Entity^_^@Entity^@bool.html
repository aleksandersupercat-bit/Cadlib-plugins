Возвращает

Возвращает объект типа LoftOptionsCheckCurvesOut, свойства которого предоставляют следующие данные:

  • Если свойство AllOpen равно true, то все поперечные сечения являются разомкнутыми.
  • Если свойство AllClosed равно true, то все поперечные сечения являются замкнутыми.
  • Если свойство AllPlanar равно true, то все поперечные сечения являются плоскими.
Описание

Метод позволяет проверить, допустимо ли сочетание указанных поперечных сечений с траекторией вытягивания, либо массивом направляющих, для успешного выполнения операции лофта. Метод выбрасывает исключение eInvalidInput, если кривые недопустимы.
