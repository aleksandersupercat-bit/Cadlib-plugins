Методы
Методы
Имя
Описание
AppendColumn
Метод добавляет в таблицу данных DataTable новый столбец.
AppendRow
Метод добавляет строку в конец данной таблицы. Дополнительно метод позволяет выполнить проверку соответствия столбцов в строке и таблице по количеству и типам данных.
Assign
Метод копирует в эту таблицу данные из указанной таблицы. Не реализован.
GetCellAt
Метод позволяет получить ячейку данных, расположенную в указанных строке и столбце.
GetColumnAt
Метод позволяет получить столбец DataColumn по его индексу в данной таблице DataTable.
GetColumnIndexAtName
Метод позволяет определить индекс столбца с указанным именем.
GetColumnNameAt
Метод позволяет получить имя столбца с указанным индексом.
GetColumnTypeAt
Метод позволяет получить тип столбца с указанным индексом.
GetRowAt
Метод позволяет получить строку с заданным индексом.
InsertColumnAt
Метод вставляет столбец в таблицу по указанному индексу столбца.
InsertRowAt
Метод вставляет в таблицу заданную строку перед строкой с указанным индексом. Дополнительно метод позволяет проверить ячейки строки на соответствие структуре таблицы.
RemoveColumnAt
Метод предназначен для удаления столбца с указанным индексом.
RemoveRowAt
Метод предназначен для удаления строки с указанным индексом.
SetCellAt
Метод позволяет задать данные в ячейке с указанными индексами строки и столбца.
SetRowAt
Метод позволяет задать (или заменить) данные строки с указанным индексом. Дополнительно метод позволяет выполнить проверки предоставленных данных на соответствие структуре таблицы.
