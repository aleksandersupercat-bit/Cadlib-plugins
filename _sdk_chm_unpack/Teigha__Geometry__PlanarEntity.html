Описание

Класс PlanarEntity является абстрактным базовым классом для всех типов плоских поверхностей в трехмерном пространстве. 

Всякая плоская поверхность S параметризуется следующим образом. 

Параметризация плоской поверхности S реализуется так, что всякая параметрическая точка с координатами u,v отображается в точку S(u,v) пространства моделирования по формуле

Копировать Код
S(u,v) = originOfPlanarEntity + (u * uAxis) + (v * vAxis)

В принципе, векторы uAxis и vAxis не обязательно должны быть нормированными или взаимно перпендикулярными, но они обязаны быть неколлинеарными. 

На любой плоской поверхности определена каноническая ортонормированная система координат, связанная с плоскостью следующим образом:

Общее уравнение плоскости имеет вид:

Копировать Код
a * X + b * Y + c * Z + d = 0

где a, b, c, d - некоторые действительные константы, удовлетворяющие условию невырожденности плоскости a^2 + b^2 + c^2 != 0 (т.е., они не могут равняться нулю одновременно). Величины a, b, c служат компонентами вектора нормали этой плоскости.
