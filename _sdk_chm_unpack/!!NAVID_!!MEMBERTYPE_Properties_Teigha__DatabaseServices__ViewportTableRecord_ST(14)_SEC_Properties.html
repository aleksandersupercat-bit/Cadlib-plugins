Свойство предназначено для включения/отключения отображения сетки в пространстве модели.

GridMajor

GridSubdivisionRestricted
Свойство предназначено для настройки разбиения ячеек на более мелкие, чем настроено в свойстве GridIncrements.

Свойство предназначено для включения/отключения режима изометрической привязки.
LowerLeftCorner
Свойство предназначено для хранения координат левой нижней точки видового экрана пространства модели.
SnapAngle
Свойство предназначено для управления углом поворота сетки и шаговой привязки для текущего видового экрана относительно текущей ПСК.
SnapBase
Свойство предназначено для управления положением начальной точки сетки и привязки для текущего видового экрана и текущей ПСК.
SnapEnabled
Свойство предназначено для управления режимом привязки.
SnapIncrements
Свойство предназначено для настройки шага привязки.
SnapPair
Свойство предназначено для изменения плоскости черчения в изометрическом режиме.
UcsFollowMode
Свойство предназначено для управления видом при переключении между ПСК в видовом экране пространства модели.
UcsSavedWithViewport
Свойство предназначено для управления взаимосвязью ПСК текущего видового экрана с ПСК остальных видовых экранов пространства модели.
UpperRightCorner
Свойство предназначено для хранения координат правой верхней точки видового экрана пространства модели.
