Свойство возвращает текущий набор выбранных в редакторе объектов.

Public ReadOnly Property Selection As SelectionSet

Возвращает

Возвращает объект типа SelectionSet, содержащий:

  • Список ObjectId выбранных объектов.
  • Дополнительную информацию о выборе (точки, матрицы преобразования — при наличии).

Если объекты не выбраны, возвращается пустой SelectionSet (с Count = 0), а не null.

Описание

Свойство предоставляет доступ к объекту типа SelectionSet, содержащему коллекцию идентификаторов (ObjectId) всех объектов, выделенных в текущий момент в чертеже. Оно доступно только для чтения и отражает состояние выборки на момент обращения к свойству — например, после завершения команды выбора или в ходе интерактивного редактирования.
