Метод предназначен для копирования предписанных свойств из исходного объекта в указанный целевой объект.

Синтаксис
C#
Visual Basic

Копировать Код
public virtual void CopyProperties( ref Entity sourceEntity, ref Entity targetEntity, int flag );

Копировать Код
Public Overridable Sub CopyProperties( ByRef sourceEntity As Entity, ByRef targetEntity As Entity, flag As Integer )

Параметры
Параметры
Описание
sourceEntity
Объект-источник свойств. Тип Entity.
targetEntity
Объект-получатель свойств. Тип Entity.
flag
Флаг, представляющий комбинацию копируемых свойств. Тип int.
Описание

Предполагается, что этот метод скопирует все предписанные свойства из объекта sourceEntity в объект targetEntity. 

Флаг flag представляет собой комбинацию битовых флагов, определяющих копируемые свойства. Перечислимый тип предоставляет в ваше распоряжение ряд предопределенных битов для аргумента flag. Поскольку flag является целым числом, неиспользуемые биты вы можете использовать по своему усмотрению.
