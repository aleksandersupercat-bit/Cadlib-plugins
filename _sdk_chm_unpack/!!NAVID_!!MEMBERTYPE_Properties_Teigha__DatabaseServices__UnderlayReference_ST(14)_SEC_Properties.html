UnderlayReference Свойства Свойства
UnderlayReference Свойства  Условные обозначения
Свойства
Имя
Описание
AdjustColorForBackground
Свойство AdjustColorForBackground позволяет определить, настроено ли содержимое подложки для текущего цвета фона.
Contrast
Свойство Contrast представляет значение контраста подложки в диапазоне [0 – 100].
ContrastLowerLimit
Свойство ContrastLowerLimit представляет нижний предел возможных значений контраста.
ContrastUpperLimit
Свойство ContrastUpperLimit представляет верхний предел возможных значений контраста.
DefaultContrast
Свойство DefaultContrast представляет значение контраста по умолчанию.
DefaultFade
Свойство DefaultFade представляет значение затухания по умолчанию.
DefinitionId
Свойство DefinitionId представляет идентификатор объекта определения подложки, на который ссылается данная подложка.
Fade
Свойство Fade представляет значение затухания для подложки в диапазоне [0 – 100].
FadeLowerLimit
Свойство FadeLowerLimit представляет нижнюю границу возможных значений затухания.
FadeUpperLimit
Свойство FadeUpperLimit представляет верхнюю границу возможных значений затухания.
Height
Свойство Height представляет высоту данной подложки на чертеже.
IsClipped
Свойство IsClipped позволяет определить , обрезан ли по границе данный объект.
IsOn
Свойство IsOn позволяет определить, отображается ли данная ссылка на подложку.
Monochrome
Свойство Monochrome позволяет определить, отображается ли содержимое подложки в монохромном режиме.
Name
Свойство Name представляет имя данного объекта.
NameOfSheet
Свойство NameOfSheet представляет имя листа.
Normal
Свойство Normal представляет вектор нормали плоскости данной подложки.
Path
Свойство Path представляет полный путь к файлу подложки.
Rotation
Свойство Rotation представляет угол поворота (в радианах) вокруг оси, определяемой точкой привязки и вектором нормали плоскости данной подложки.
Transform
Свойство Transform представляет матрицу преобразования данной подложки.
UnderlayLayerCollection
Свойство UnderlayLayerCollection представляет коллекцию слоев подложки в базе данных.
Width
Свойство Width представляет ширину данной подложки на чертеже.
position
Свойство Position представляет положение подложки в мировой системе координат (WCS).
scaleFactors
Свойство ScaleFactors представляет коэффициенты масштабирования данной подложки.
